# Паттерны xbsl-file-add

## P1 — Стандартные вложения

Реквизит `Файлы` встроен во все Справочники и Документы — новые реквизиты не добавляются.

### Компонент формы

```yaml
- Тип: СписокФайлов
  Имя: Файлы
  Значение: =Объект.Файлы
  МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 50)
```

---

## P2 — Одиночный файл

### Реквизит объекта

```yaml
- Ид: <UUID>
  Имя: <ИмяРеквизита>
  Тип: ДвоичныйОбъект.Ссылка?
```

### Компонент формы

```yaml
- Тип: СписокФайлов
  Имя: <ИмяРеквизита>
  Значение: =Объект.<ИмяРеквизита>
  МаксимальноеКоличествоФайлов: 1
  МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 20)
```

Опциональные параметры (добавлять при необходимости):

```yaml
  Фильтр:
    - application/pdf
    - .docx
  МинимальноеКоличествоФайлов: 1
  ТекстКнопкиЗагрузки: Загрузить файл
```

---

## P3 — Категория файлов

### Реквизит объекта

```yaml
- Ид: <UUID>
  Имя: <ИмяКатегории>
  Тип: Массив<ДвоичныйОбъект.Ссылка>
```

### Компонент формы

```yaml
- Тип: СписокФайлов
  Имя: <ИмяКатегории>
  Значение: =Объект.<ИмяКатегории>
  МаксимальноеКоличествоФайлов: 5
  МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 50)
```

Опциональные параметры:

```yaml
  МаксимальныйРазмерФайла: =новый РазмерБайтов(Мб = 10)
  МинимальноеКоличествоФайлов: 1
  Фильтр:
    - application/pdf
    - .docx
    - .xlsx
  ТекстКнопкиЗагрузки: Добавить файл
```

## P4 — Несколько категорий

Повторить шаблон P3 для каждой категории (отдельный реквизит + отдельный компонент).

### Реквизиты объекта

```yaml
- Ид: <UUID-1>
  Имя: Договор
  Тип: Массив<ДвоичныйОбъект.Ссылка>
- Ид: <UUID-2>
  Имя: ТехническоеЗадание
  Тип: Массив<ДвоичныйОбъект.Ссылка>
- Ид: <UUID-3>
  Имя: Акт
  Тип: Массив<ДвоичныйОбъект.Ссылка>
```

### Компоненты формы (один блок на каждую категорию)

```yaml
- Тип: СписокФайлов
  Имя: Договор
  Значение: =Объект.Договор
  МаксимальноеКоличествоФайлов: 3
  МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 30)
  Фильтр:
    - application/pdf
    - .docx
- Тип: СписокФайлов
  Имя: ТехническоеЗадание
  Значение: =Объект.ТехническоеЗадание
  МаксимальноеКоличествоФайлов: 1
  МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 10)
- Тип: СписокФайлов
  Имя: Акт
  Значение: =Объект.Акт
  МаксимальноеКоличествоФайлов: 5
  МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 50)
```

---

## Справка по параметрам СписокФайлов

| Параметр | Тип | Описание |
|----------|-----|---------|
| `Значение` | `Массив<ДвоичныйОбъект.Ссылка>` | Привязка к реквизиту объекта |
| `МаксимальноеКоличествоФайлов` | `Число` | Максимально допустимое количество файлов |
| `МинимальноеКоличествоФайлов` | `Число` | Минимальное количество (валидация) |
| `МаксимальныйРазмерФайла` | `РазмерБайтов` | Максимальный размер каждого файла |
| `МаксимальныйСуммарныйРазмерФайлов` | `РазмерБайтов` | Максимальный суммарный размер всех файлов |
| `Фильтр` | `Массив<Строка>` | Разрешённые MIME-типы и расширения |
| `ТекстКнопкиЗагрузки` | `Строка` | Текст кнопки добавления файлов |

### РазмерБайтов — конструктор

```yaml
=новый РазмерБайтов(Мб = 50)     # 50 МБ
=новый РазмерБайтов(Гб = 1)      # 1 ГБ
=новый РазмерБайтов(Кб = 512)    # 512 КБ
```
