# Справочник элементов UI форм

## ПолеВвода

```yaml
Тип: ПолеВвода<<Тип>>
Имя: <ИмяЭлемента>              # опционально — нужно если на него ссылаются из кода
Заголовок: $ЛокализованныеСтроки.<Ключ>
Значение: =Объект.<Поле>
Обязательное: Истина             # опционально
СообщениеОшибка: <Текст>         # опционально, показывается при незаполненном обязательном поле
Видимость: =<Выражение>          # опционально — скрыть/показать по условию
Доступность: =<Выражение>        # опционально — заблокировать редактирование
ОтслеживатьИзменениеДанных: Ложь # для полей, не влияющих на признак изменения
ШиринаВКолонках: Двойная         # опционально — растянуть поле
```

**Типы для ПолеВвода:**
| Тип | Пример |
|-----|--------|
| `Строка` | Наименование, ФИО, адрес |
| `Число` | Сумма, Количество |
| `Булево` | Через Флажок, не ПолеВвода |
| `Дата` | Дата без времени |
| `ДатаВремя` | Дата и время |
| `Момент` | Временная метка |
| `<Справочник>.Ссылка?` | Ссылка на справочник |
| `<Перечисление>` | Перечисление (обязательное) |
| `<Перечисление>?` | Перечисление (необязательное) |
| `Пользователи.Ссылка?` | Системный пользователь |

**Многострочная строка:**
```yaml
Тип: ПолеВвода<Строка>
Значение: =Объект.Описание
НастройкиВводаСтроки:
    Многострочная: Истина
```

**Фильтр по ссылочному полю:**
```yaml
Тип: ПолеВвода<КонтактныеЛица.Ссылка?>
Значение: =Объект.КонтактноеЛицо
НастройкиВводаСсылок:
    НастройкиПоТипу:
        -
            Ключ: КонтактныеЛица.Ссылка
            Значение:
                Фильтр:
                    Элементы:
                        -
                            Тип: ЭлементФильтра
                            ВидСравнения: Равно
                            Значение: =Объект.<РодительскоеПоле>
                            Поле: <ПолеФильтрации>
```

## Флажок (булево)

```yaml
Тип: Флажок
Заголовок: $ЛокализованныеСтроки.<Ключ>
Значение: =Объект.<БулевоПоле>
Вид: Переключатель    # или Флажок (по умолчанию)
Видимость: =<Выражение>
```

---

## Надпись (только для отображения)

```yaml
Тип: Надпись
Заголовок: <Заголовок поля>
Значение: =ДанныеСтроки.Данные.<Поле>
Изображение: <Иконка>.svg          # опционально
Видимость: =не ДанныеСтроки.Данные.<Поле>.Пусто()
РастягиватьПоГоризонтали: Ложь
```

## Группа (контейнер)

```yaml
Тип: Группа
Компоновка: Вертикальная    # или Горизонтальная (по умолчанию)
ШиринаВКолонках: Двойная
ИнтервалМеждуЭлементамиПоВертикали: Отсутствует   # опционально
Содержимое:
    - ...
```

## СворачиваемыйКомпонент (раскрывающийся раздел)

```yaml
Тип: СворачиваемыйКомпонент
Заголовок: $ЛокализованныеСтроки.<Ключ>
РастягиватьПоГоризонтали: Истина
ШиринаВКолонках: Четверная
Содержимое:
    Тип: Группа
    Содержимое:
        - ...
```

## СписокФайлов

```yaml
Тип: СписокФайлов
Имя: Файлы
Заголовок: $ЛокализованныеСтроки.Файлы
Значение: =Объект.Файлы
МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 50)
Фильтр:                    # опционально — разрешённые типы
    - application/pdf
    - .docx
    - image/*
```

## Картинка

```yaml
Тип: Картинка
Имя: Фото
Видимость: =Объект.Фото != Неопределено
Изображение: =Объект.Фото
МаксимальнаяВысота: 200
РастягиватьПоГоризонтали: Ложь
ШиринаВКолонках: Одинарная
```

## Таблица (для табличной части)

```yaml
Тип: Таблица<ИсточникДанныхМассив<<ИмяОбъекта>.<ИмяТЧ>>>
Имя: <ИмяТЧ>
Навигация: ПодгрузкаПриПрокрутке
ОтображатьНумерациюСтрок: Ложь
Источник:
    Данные: =Объект.<ИмяТЧ>
Колонки:
    -
        Тип: СтандартнаяКолонкаТаблицы<<ИмяОбъекта>.<ИмяТЧ>>
        Заголовок: <Заголовок>
        ПолеЗначения: <ИмяКолонки>
        НастройкиРедактирования:
            Тип: НастройкиРедактированияПоляВвода
            ПриИзменении: <ОбработчикИзменения>   # опционально
Команды:
    Тип: ФрагментКомандногоИнтерфейса
    Элементы:
        - =Компоненты.<ИмяТЧ>.ДобавитьСтроку
КомандыСтроки:
    Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<<ИмяОбъекта>.<ИмяТЧ>>>
    Элементы:
        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВверх
        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВниз
        - =Компоненты.<ИмяТЧ>.Удалить
```

## Пустое состояние списка

```yaml
ПустыеСостояния:
    Тип: ПредставленияПустыхСостоянийСписка
    НетРезультатовФильтрации:
        Тип: ПредставлениеПустогоСостоянияСписка
        Заголовок: Ничего не найдено
        Изображение:
            Тип: ДвоичныйОбъект.Ссылка
            Значение: ПустойСписок.svg
        Пояснение: Нет записей, соответствующих условиям
```

## Ширина в колонках

| Значение | Смысл |
|----------|-------|
| `Одинарная` | 1/4 ширины формы |
| `Двойная` | 1/2 ширины |
| `Тройная` | 3/4 ширины |
| `Четверная` | На всю ширину |

---

## Привязки данных (выражения)

| Выражение | Что означает |
|-----------|-------------|
| `=Объект.Поле` | Поле текущего объекта |
| `=МоёСвойство` | Свойство формы |
| `=ДанныеСтроки.Данные.Поле` | Поле строки в таблице/списке |
| `=ЭтоНовый()` | Истина если запись ещё не сохранена |
| `=КлючОбъекта` | Ссылка на текущий объект |
| `'=ЭтоНовый() ? Вариант1 : Вариант2'` | Тернарный оператор в кавычках |
| `=не Объект.ПометкаУдаления` | Логическое отрицание |
