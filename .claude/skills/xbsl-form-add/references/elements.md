# Справочник элементов UI форм

## ПолеВвода

```yaml
Тип: ПолеВвода<<Тип>>
Имя: <ИмяЭлемента>              # нужно если на него ссылаются из кода
Значение: =Объект.<Поле>
Видимость: =<Выражение>          # опционально
Доступность: =<Выражение>        # опционально
```

**Типы для ПолеВвода:**
| Тип | Пример |
|-----|--------|
| `Строка` | Наименование, ФИО, адрес |
| `Число` | Сумма, Количество |
| `Дата` | Дата без времени |
| `ДатаВремя` | Дата и время |
| `<Справочник>.Ссылка?` | Ссылка на справочник |
| `<Перечисление>` | Перечисление (обязательное) |
| `<Перечисление>?` | Перечисление (необязательное) |

**Многострочная строка:**
```yaml
Тип: ПолеВвода<Строка>
Значение: =Объект.Описание
НастройкиВводаСтроки:
    Многострочная: Истина
```

## Флажок (булево)

```yaml
Тип: Флажок
Значение: =Объект.<БулевоПоле>
Вид: Переключатель    # или Флажок (по умолчанию)
```

## Группа (контейнер)

```yaml
Тип: Группа
Компоновка: Вертикальная    # или Горизонтальная (по умолчанию)
ШиринаВКолонках: Двойная
Содержимое:
    - ...
```

---

## СворачиваемыйКомпонент

```yaml
Тип: СворачиваемыйКомпонент
Заголовок: <Заголовок>
РастягиватьПоГоризонтали: Истина
Содержимое:
    Тип: Группа
    Содержимое:
        - ...
```

МаксимальныйСуммарныйРазмерФайлов: =новый РазмерБайтов(Мб = 50)

## Картинка

```yaml
Тип: Картинка
Изображение: =Объект.Фото ?? Ресурс{ПустойАватар.svg}.Ссылка
МаксимальнаяВысота: 200
Масштабирование: Пропорционально
```

---

## Надпись (read-only текст/значение)

```yaml
Тип: Надпись
Заголовок: <Статический текст>      # или
Значение: =Объект.<Поле>            # динамическое значение
ОценкаИнформации: Положительная     # Положительная | Предупреждение | Отрицательная | Обычная
Цвет: =...                          # опционально
```

Используй `Надпись` для read-only отображения вычисляемых значений, статусов, подписей.

---

## Кнопка (action button)

```yaml
Тип: Кнопка
Заголовок: <Текст>
Изображение: <иконка>.svg
Команды: =<ИмяКоманды>
ОпасностьДействия: Высокая          # Отсутствует | Средняя | Высокая
```

Используй для кастомных кнопок действий внутри формы. Для стандартных команд формы (`ЗаписатьИЗакрыть`, `Удалить` и т.д.) используй `ОсновнаяКоманда` / `ДополнительныеКоманды`.

---

## Страницы / Страница (вкладки)

```yaml
Тип: Страницы
РастягиватьПоВертикали: Истина
Страницы:
    -
        Тип: Страница
        Заголовок: Основное
        Содержимое:
            Тип: Группа
            Содержимое:
                - ...
    -
        Тип: Страница
        Заголовок: Дополнительно
        Содержимое:
            Тип: Группа
            Содержимое:
                - ...
```

Используй `Страницы` как альтернативу `ШаблонФормыСРазделами` когда нужен вкладочный layout внутри произвольного шаблона.

## Ширина в колонках

| Значение | Смысл |
|----------|-------|
| `Одинарная` | 1/4 ширины формы |
| `Двойная` | 1/2 ширины |
| `Тройная` | 3/4 ширины |
| `Четверная` | На всю ширину |

---

## Привязки данных

| Выражение | Что означает |
|-----------|-------------|
| `=Объект.Поле` | Поле текущего объекта |
| `=ДанныеСтроки.Данные.Поле` | Поле строки в ФормаСписка |
| `=МоёСвойство` | Свойство формы |
| `=ЭтоНовый()` | Истина если запись не сохранена |
| `=Объект.Поле ?? Ресурс{икона.svg}.Ссылка` | С фолбэком |
