# Спецификация: ФормаОбъекта

## Дефолты

- **Имя** — `{ИмяОбъекта}ФормаОбъекта` (PascalCase)
- **Видимость** — `ВПодсистеме`
- **UUID** — ровно 1 на форму
- **ВключатьВАвтоИнтерфейс** — всегда `Ложь`
- **Шаблон** — `ПроизвольныйШаблонФормы` для объектов без ТЧ, `ШаблонФормыСРазделами` если есть табличные части
- **Layout в ШаблонФормыСРазделами** — `РазделФормы` (вкладки) если ТЧ несколько или полей мало; `Группа` (панели) если 1 ТЧ и 5+ реквизитов

## Структура YAML

### Справочник (без табличных частей)

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОбъекта<<ИмяОбъекта>.Объект>
    ВключатьВАвтоИнтерфейс: Ложь
    Заголовок: <Заголовок>
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Удалить
            - =Восстановить
            # - =СоздатьКопию        # добавь если нужна кнопка "Создать копию"
            # - =УдалитьНавсегда     # добавь если нужно безвозвратное удаление
    ОсновнаяКоманда: =ЗаписатьИЗакрыть
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        ШиринаВКолонках: Одинарная
        Содержимое:
            Тип: Группа
            Содержимое:
                -
                    Тип: ПолеВвода<Строка>
                    Имя: Наименование
                    Значение: =Объект.Наименование
                -
                    Тип: ПолеВвода<<Тип>>
                    Имя: <ИмяПоля>
                    Значение: =Объект.<ИмяПоля>
```

### Объект с 1 ТЧ и 5+ реквизитами — панельный layout (реквизиты слева, ТЧ справа)

Используй для любого объекта (Справочник или Документ) с одной табличной частью и пятью и более реквизитами.
`ОсновнойРаздел` и `ДополнительныеРазделы` — тип `Группа` (не `РазделФормы`): это создаёт горизонтальные панели вместо вкладок.

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОбъекта<<ИмяОбъекта>.Объект>
    ВключатьВАвтоИнтерфейс: Ложь
    Заголовок: <Заголовок>
    РастягиватьПоВертикали: Истина
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Удалить
            - =Восстановить
    ОсновнаяКоманда: =ЗаписатьИЗакрыть
    Содержимое:
        Тип: ШаблонФормыСРазделами
        РастягиватьПоВертикали: Истина
        ОсновнойРаздел:
            Тип: Группа                  # ← Группа, не РазделФормы — это даёт панели
            Заголовок: Основное
            ШиринаВКолонках: Одинарная   # левая колонка, 25% ширины
            Содержимое:
                -
                    Тип: ПолеВвода<<Тип1>>
                    Имя: <ИмяПоля1>
                    Значение: =Объект.<ИмяПоля1>
                -
                    Тип: ПолеВвода<<Тип2>>
                    Имя: <ИмяПоля2>
                    Значение: =Объект.<ИмяПоля2>
                # ... все реквизиты объекта
        ДополнительныеРазделы:
            -
                Тип: Группа              # ← Группа, не РазделФормы
                Заголовок: <ИмяТЧ>
                ШиринаВКолонках: Двойная # правая панель, 50% ширины
                Содержимое:
                    -
                        Тип: СворачиваемыйКомпонент
                        Заголовок: <ИмяТЧ>
                        РастягиватьПоГоризонтали: Истина
                        ШиринаВКолонках: Четверная
                        Содержимое:
                            Тип: Таблица<ИсточникДанныхМассив<<ИмяОбъекта>.<ИмяТЧ>>>
                            Имя: <ИмяТЧ>
                            ВидОтображенияКомандСтроки: ПриНаведении
                            ОтображатьНумерациюСтрок: Ложь
                            РастягиватьПоВертикали: Ложь
                            ШиринаВКолонках: Четверная
                            Источник:
                                Данные: =Объект.<ИмяТЧ>
                            Колонки:
                                -
                                    Тип: СтандартнаяКолонкаТаблицы<<ИмяОбъекта>.<ИмяТЧ>>
                                    Заголовок: <Заголовок>
                                    ПолеЗначения: <ИмяКолонки>
                                    НастройкиРедактирования:
                                        Тип: НастройкиРедактированияПоляВвода
                            Команды:
                                Тип: ФрагментКомандногоИнтерфейса
                                Элементы:
                                    - =Компоненты.<ИмяТЧ>.ДобавитьСтроку
                            КомандыСтроки:
                                Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<<ИмяОбъекта>.<ИмяТЧ>>>
                                Элементы:
                                    - =Компоненты.<ИмяТЧ>.Удалить
```

> Если есть вторичные реквизиты (Ответственный, даты, файлы и т.п.) — добавь второй `СворачиваемыйКомпонент` "Дополнительно" в ту же правую `Группу` после таблицы.

### Объект с ТЧ — вкладочный layout (2+ ТЧ или менее 5 реквизитов)

```yaml
    Заголовок: <Заголовок>
    РастягиватьПоВертикали: Истина
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Удалить
            - =Восстановить
    ОсновнаяКоманда: =ЗаписатьИЗакрыть

        ОсновнойРаздел:
            Тип: РазделФормы
            Заголовок: Основное
            ШиринаВКолонках: Одинарная
            Содержимое:
                -
                    Имя: ГруппаПолей
                    Содержимое:
                        -
                            Тип: ПолеВвода<Строка>

                        Содержимое:
                            -
                                Тип: Таблица<ИсточникДанныхМассив<<ИмяОбъекта>.<ИмяТЧ>>>
                                Имя: <ИмяТЧ>
                                Источник:
                                    Данные: =Объект.<ИмяТЧ>

                                    -
                                        Тип: СтандартнаяКолонкаТаблицы<<ИмяОбъекта>.<ИмяТЧ>>

                                        ПолеЗначения: <ИмяКолонки>
                                        НастройкиРедактирования:
                                            Тип: НастройкиРедактированияПоляВвода
                                Команды:
                                    Тип: ФрагментКомандногоИнтерфейса
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ДобавитьСтроку
                                КомандыНадОтмеченнымиСтроками:
                                    Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<Массив<<ИмяОбъекта>.<ИмяТЧ>>>>
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВверх
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВниз
                                        - =Компоненты.<ИмяТЧ>.Удалить
```

## Маппинг типов полей

Определяй тип `ПолеВвода` по типу поля из YAML объекта:

| Тип поля в YAML | Элемент формы |
|-----------------|---------------|
| `Строка` | `ПолеВвода<Строка>` |
| `Число` | `ПолеВвода<Число>` |
| `Дата` | `ПолеВвода<Дата>` |
| `ДатаВремя` | `ПолеВвода<ДатаВремя>` |
| `Булево` | `Флажок` |
| `<Справочник>.Ссылка` | `ПолеВвода<<Справочник>.Ссылка?>` |
| `<Перечисление>` | `ПолеВвода<<Перечисление>?>` |
| `Строка` (Описание/Комментарий/длинный текст) | `ПолеВвода<Строка>` + `НастройкиВводаСтроки: Многострочная: Истина` |

## Правила

- `ВключатьВАвтоИнтерфейс: Ложь` — обязательно всегда
- `ОсновнаяКоманда: =ЗаписатьИЗакрыть` — основная кнопка
- `ОбычныеКоманды: =Записать` — добавляй только если явно нужна кнопка "Записать" без закрытия
- Для Справочника первое поле — `Наименование`; для Документа — `Номер` + `Дата`
- `РастягиватьПоВертикали: Истина` — нужно для форм с ТЧ, чтобы таблица занимала всю высоту
- **Выбор layout для ТЧ:**
  - 1 ТЧ + 5 и более реквизитов → панели: `ОсновнойРаздел: Тип: Группа` (Одинарная) + `ДополнительныеРазделы: Тип: Группа` (Двойная) с `СворачиваемыйКомпонент`
  - 2+ ТЧ или менее 5 реквизитов → вкладки: `ОсновнойРаздел: Тип: РазделФормы` + `ДополнительныеРазделы: Тип: РазделФормы`
- **Свойства таблицы в панельном layout:** `ВидОтображенияКомандСтроки: ПриНаведении`, `ОтображатьНумерациюСтрок: Ложь`, `РастягиватьПоВертикали: Ложь`
- Свойства формы (реактивное состояние) добавляй в секцию `Свойства` только если нужна интерактивность

## Свойства формы (реактивное состояние)

```yaml
Свойства:
    -
        Имя: <ИмяСвойства>
        Тип: Булево
        ЗначениеПоУмолчанию: Ложь
    -
        Имя: <ИмяСвойства>
        Тип: <ТипПеречисления>?
```
