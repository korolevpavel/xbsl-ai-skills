# Спецификация: ФормаОбъекта

## Общая структура

Все формы ФормаОбъекта используют одинаковый заголовок. Выбор варианта `Содержимое` — по `suggested_layout` из `form_info.py`.

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОбъекта<<ИмяОбъекта>.Объект>
    ВключатьВАвтоИнтерфейс: Ложь
    Заголовок: <Заголовок>
    РастягиватьПоВертикали: Истина  # только если есть ТЧ
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Удалить
            - =Восстановить
            # - =СоздатьКопию
            # - =УдалитьНавсегда
    ОсновнаяКоманда: =ЗаписатьИЗакрыть
    Содержимое:
        <см. варианты ниже>
```

---

## Вариант: simple (нет ТЧ)

```yaml
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        ШиринаВКолонках: Одинарная
        Содержимое:
            Тип: Группа
            Содержимое:
                -
                    Тип: ПолеВвода<Строка>
                    Имя: Наименование
                    Значение: =Объект.Наименование
                -
                    Тип: ПолеВвода<<Тип>>
                    Имя: <ИмяПоля>
                    Значение: =Объект.<ИмяПоля>
```

---

## Вариант: panels (1 ТЧ, 5+ реквизитов)

Реквизиты слева (Одинарная), ТЧ справа в `СворачиваемыйКомпонент` (Двойная).
`ОсновнойРаздел` и `ДополнительныеРазделы` — тип `Группа` (не `РазделФормы`).

```yaml
    Содержимое:
        Тип: ШаблонФормыСРазделами
        РастягиватьПоВертикали: Истина
        ОсновнойРаздел:
            Тип: Группа
            Заголовок: Основное
            ШиринаВКолонках: Одинарная
            Содержимое:
                -
                    Тип: ПолеВвода<<Тип1>>
                    Имя: <ИмяПоля1>
                    Значение: =Объект.<ИмяПоля1>
                # ... все реквизиты объекта
        ДополнительныеРазделы:
            -
                Тип: Группа
                Заголовок: <ИмяТЧ>
                ШиринаВКолонках: Двойная
                Содержимое:
                    -
                        Тип: СворачиваемыйКомпонент
                        Заголовок: <ИмяТЧ>
                        РастягиватьПоГоризонтали: Истина
                        ШиринаВКолонках: Четверная
                        Содержимое:
                            Тип: Таблица<ИсточникДанныхМассив<<ИмяОбъекта>.<ИмяТЧ>>>
                            Имя: <ИмяТЧ>
                            ВидОтображенияКомандСтроки: ПриНаведении
                            ОтображатьНумерациюСтрок: Ложь
                            РастягиватьПоВертикали: Ложь
                            ШиринаВКолонках: Четверная
                            Источник:
                                Данные: =Объект.<ИмяТЧ>
                            Колонки:
                                -
                                    Тип: СтандартнаяКолонкаТаблицы<<ИмяОбъекта>.<ИмяТЧ>>
                                    Заголовок: <Заголовок>
                                    ПолеЗначения: <ИмяКолонки>
                                    НастройкиРедактирования:
                                        Тип: НастройкиРедактированияПоляВвода
                            Команды:
                                Тип: ФрагментКомандногоИнтерфейса
                                Элементы:
                                    - =Компоненты.<ИмяТЧ>.ДобавитьСтроку
                            КомандыСтроки:
                                Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<<ИмяОбъекта>.<ИмяТЧ>>>
                                Элементы:
                                    - =Компоненты.<ИмяТЧ>.Удалить
```

> Если есть вторичные реквизиты (Ответственный, даты и т.п.) — добавь второй `СворачиваемыйКомпонент` «Дополнительно» в ту же правую `Группу` после таблицы.

## Вариант: tabs (2+ ТЧ или менее 5 реквизитов)

`ОсновнойРаздел` и `ДополнительныеРазделы` — тип `РазделФормы`.

```yaml
    Содержимое:
        Тип: ШаблонФормыСРазделами
        РастягиватьПоВертикали: Истина
        ОсновнойРаздел:
            Тип: РазделФормы
            Заголовок: Основное
            ШиринаВКолонках: Одинарная
            Содержимое:
                -
                    Имя: ГруппаПолей
                    Содержимое:
                        -
                            Тип: ПолеВвода<Строка>
                            Имя: Номер
                            Значение: =Объект.Номер
                        -
                            Тип: ПолеВвода<ДатаВремя>
                            Имя: Дата
                            Значение: =Объект.Дата
                        -
                            Тип: ПолеВвода<<Справочник>.Ссылка?>
                            Имя: <ИмяПоля>
                            Значение: =Объект.<ИмяПоля>
        ДополнительныеРазделы:
            -
                Тип: РазделФормы
                Заголовок: <ИмяТЧ>
                Содержимое:
                    -
                        Содержимое:
                            -
                                Тип: Таблица<ИсточникДанныхМассив<<ИмяОбъекта>.<ИмяТЧ>>>
                                Имя: <ИмяТЧ>
                                Источник:
                                    Данные: =Объект.<ИмяТЧ>
                                Колонки:
                                    -
                                        Тип: СтандартнаяКолонкаТаблицы<<ИмяОбъекта>.<ИмяТЧ>>
                                        Заголовок: <Заголовок>
                                        ПолеЗначения: <ИмяКолонки>
                                        НастройкиРедактирования:
                                            Тип: НастройкиРедактированияПоляВвода
                                Команды:
                                    Тип: ФрагментКомандногоИнтерфейса
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ДобавитьСтроку
                                КомандыНадОтмеченнымиСтроками:
                                    Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<Массив<<ИмяОбъекта>.<ИмяТЧ>>>>
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВверх
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВниз
                                        - =Компоненты.<ИмяТЧ>.Удалить
```

## Маппинг типов полей

| Тип поля в YAML | Элемент формы |
|-----------------|---------------|
| `Строка` | `ПолеВвода<Строка>` |
| `Число` | `ПолеВвода<Число>` |
| `Дата` | `ПолеВвода<Дата>` |
| `ДатаВремя` | `ПолеВвода<ДатаВремя>` |
| `Булево` | `Флажок` |
| `<Справочник>.Ссылка` | `ПолеВвода<<Справочник>.Ссылка?>` |
| `<Перечисление>` | `ПолеВвода<<Перечисление>?>` |
| `Строка` (Описание/Комментарий) | `ПолеВвода<Строка>` + `НастройкиВводаСтроки: Многострочная: Истина` |

## Правила

- `РастягиватьПоВертикали: Истина` — добавлять в `Наследует` только для форм с ТЧ
- Для Справочника первое поле — `Наименование`; для Документа — `Номер` + `Дата`
- Таблица в panels: `ВидОтображенияКомандСтроки: ПриНаведении`, `ОтображатьНумерациюСтрок: Ложь`, `РастягиватьПоВертикали: Ложь`
- `ОбычныеКоманды: =Записать` — добавляй только если явно нужна кнопка «Записать» без закрытия
- **Иерархический справочник**: если `form_info.py` вернул `is_hierarchical: true` (паттерн А), добавь поле `Родитель` в форму объекта. Реквизит системный — в YAML объекта не записан, `form_info.py` его не возвращает. Размести сразу после `Наименование`:
  ```yaml
  -
      Тип: ПолеВвода<<ИмяСправочника>.Ссылка?>
      Имя: Родитель
      Значение: =Объект.Родитель
  ```
  Для паттерна Б (`additional_hierarchies` не пуст) поле `Родитель` не добавляется — связь через обычные реквизиты-ссылки, они уже есть в `fields`.
