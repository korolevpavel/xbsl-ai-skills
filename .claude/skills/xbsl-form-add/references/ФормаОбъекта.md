# Спецификация: ФормаОбъекта

## Дефолты

- **Имя** — `{ИмяОбъекта}ФормаОбъекта` (PascalCase)
- **Видимость** — `ВПодсистеме`
- **UUID** — ровно 1 на форму
- **ВключатьВАвтоИнтерфейс** — всегда `Ложь`
- **Шаблон** — `ПроизвольныйШаблонФормы` для объектов без ТЧ, `ШаблонФормыСРазделами` если есть табличные части

## Структура YAML

### Справочник (без табличных частей)

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОбъекта<<ИмяОбъекта>.Объект>
    ВключатьВАвтоИнтерфейс: Ложь
    Заголовок: <Заголовок>
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Удалить
            - =Восстановить
    ОсновнаяКоманда: =ЗаписатьИЗакрыть
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        ШиринаВКолонках: Одинарная
        Содержимое:
            Тип: Группа
            Содержимое:
                -
                    Тип: ПолеВвода<Строка>
                    Имя: Наименование
                    Значение: =Объект.Наименование
                -
                    Тип: ПолеВвода<<Тип>>
                    Имя: <ИмяПоля>
                    Значение: =Объект.<ИмяПоля>
```

### Документ (с табличными частями)

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОбъекта<<ИмяОбъекта>.Объект>
    ВключатьВАвтоИнтерфейс: Ложь
    Заголовок: <Заголовок>
    РастягиватьПоВертикали: Истина
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Удалить
            - =Восстановить
    ОсновнаяКоманда: =ЗаписатьИЗакрыть
    Содержимое:
        Тип: ШаблонФормыСРазделами
        РастягиватьПоВертикали: Истина
        ОсновнойРаздел:
            Тип: РазделФормы
            Заголовок: Основное
            ШиринаВКолонках: Одинарная
            Содержимое:
                -
                    Имя: ГруппаПолей
                    Содержимое:
                        -
                            Тип: ПолеВвода<Строка>
                            Имя: Номер
                            Значение: =Объект.Номер
                        -
                            Тип: ПолеВвода<ДатаВремя>
                            Имя: Дата
                            Значение: =Объект.Дата
                        -
                            Тип: ПолеВвода<<Справочник>.Ссылка?>
                            Имя: <ИмяПоля>
                            Значение: =Объект.<ИмяПоля>
        ДополнительныеРазделы:
            -
                Тип: РазделФормы
                Заголовок: <ИмяТЧ>
                Содержимое:
                    -
                        Содержимое:
                            -
                                Тип: Таблица<ИсточникДанныхМассив<<ИмяОбъекта>.<ИмяТЧ>>>
                                Имя: <ИмяТЧ>
                                Источник:
                                    Данные: =Объект.<ИмяТЧ>
                                Колонки:
                                    -
                                        Тип: СтандартнаяКолонкаТаблицы<<ИмяОбъекта>.<ИмяТЧ>>
                                        Заголовок: <Заголовок>
                                        ПолеЗначения: <ИмяКолонки>
                                        НастройкиРедактирования:
                                            Тип: НастройкиРедактированияПоляВвода
                                Команды:
                                    Тип: ФрагментКомандногоИнтерфейса
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ДобавитьСтроку
                                КомандыНадОтмеченнымиСтроками:
                                    Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<Массив<<ИмяОбъекта>.<ИмяТЧ>>>>
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВверх
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВниз
                                        - =Компоненты.<ИмяТЧ>.Удалить
```

## Маппинг типов полей

Определяй тип `ПолеВвода` по типу поля из YAML объекта:

| Тип поля в YAML | Элемент формы |
|-----------------|---------------|
| `Строка` | `ПолеВвода<Строка>` |
| `Число` | `ПолеВвода<Число>` |
| `Дата` | `ПолеВвода<Дата>` |
| `ДатаВремя` | `ПолеВвода<ДатаВремя>` |
| `Булево` | `Флажок` |
| `<Справочник>.Ссылка` | `ПолеВвода<<Справочник>.Ссылка?>` |
| `<Перечисление>` | `ПолеВвода<<Перечисление>?>` |
| `Строка` (Описание/Комментарий/длинный текст) | `ПолеВвода<Строка>` + `НастройкиВводаСтроки: Многострочная: Истина` |

## Правила

- `ВключатьВАвтоИнтерфейс: Ложь` — обязательно всегда
- `ОсновнаяКоманда: =ЗаписатьИЗакрыть` — основная кнопка
- `ОбычныеКоманды: =Записать` — добавляй только если явно нужна кнопка "Записать" без закрытия
- Для Справочника первое поле — `Наименование`; для Документа — `Номер` + `Дата`
- `РастягиватьПоВертикали: Истина` — нужно для форм с ТЧ, чтобы таблица занимала всю высоту
- Свойства формы (реактивное состояние) добавляй в секцию `Свойства` только если нужна интерактивность

## Свойства формы (реактивное состояние)

```yaml
Свойства:
    -
        Имя: <ИмяСвойства>
        Тип: Булево
        ЗначениеПоУмолчанию: Ложь
    -
        Имя: <ИмяСвойства>
        Тип: <ТипПеречисления>?
```
