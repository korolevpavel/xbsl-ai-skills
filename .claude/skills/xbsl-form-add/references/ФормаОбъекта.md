# Спецификация: ФормаОбъекта

## Дефолты

- **Имя** — `{ИмяОбъекта}ФормаОбъекта` (PascalCase)
- **Видимость** — `ВПодсистеме`. Менять на `ВПроекте` если форма открывается из других подсистем
- **UUID** — ровно 1 на форму; элементы (поля, колонки) UUID не требуют
- **Шаблон** — `ПроизвольныйШаблонФормы` для простых объектов, `ШаблонФормыСРазделами` если есть табличные части

## UUID

Нужен: 1 (сам КомпонентИнтерфейса).

## Структура YAML

### Минимальная (Справочник без ТЧ):

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОбъекта<<ИмяОбъекта>.Объект>
    Заголовок: $ЛокализованныеСтроки.<КлючПредставленияЕд>
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Обновить
            - =СоздатьКопию
            - =Удалить
            - =Восстановить
    ОбычныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Записать
    ОсновнаяКоманда: =ЗаписатьИЗакрыть
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        ШиринаВКолонках: Одинарная
        Содержимое:
            Тип: Группа
            Содержимое:
                -
                    Тип: ПолеВвода<Строка>
                    Заголовок: $ЛокализованныеСтроки.<КлючНаименование>
                    Значение: =Объект.Наименование
                    Обязательное: Истина
                    СообщениеОшибка: Введите наименование
                -
                    Ид: <UUID>
                    Тип: ПолеВвода<<Тип>>
                    Заголовок: $ЛокализованныеСтроки.<КлючПоля>
                    Значение: =Объект.<ИмяПоля>
```

### С табличными частями (Документ):

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОбъекта<<ИмяОбъекта>.Объект>
    Заголовок: =ЗаголовокФормы()          # или $ЛокализованныеСтроки.<Ключ>
    РастягиватьПоВертикали: Истина
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Удалить
            - =Восстановить
    ОбычныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Записать
    ОсновнаяКоманда: =ЗаписатьИЗакрыть
    Содержимое:
        Тип: ШаблонФормыСРазделами
        РастягиватьПоВертикали: Истина
        ШиринаВКолонках: Двойная
        ОсновнойРаздел:
            Тип: РазделФормы
            Заголовок: Основное
            Содержимое:
                -
                    Тип: Группа
                    Содержимое:
                        -
                            Тип: ПолеВвода<Строка>
                            Имя: Номер
                            Значение: =Объект.Номер
                        -
                            Тип: ПолеВвода<ДатаВремя>
                            Имя: Дата
                            Значение: =Объект.Дата
                        -
                            Тип: ПолеВвода<<Справочник>.Ссылка?>
                            Заголовок: <Заголовок>
                            Значение: =Объект.<Поле>
                            Обязательное: Истина
                            СообщениеОшибка: <Текст>
        ДополнительныеРазделы:
            -
                Тип: РазделФормы
                Заголовок: <ИмяТЧ>
                Содержимое:
                    -
                        Содержимое:
                            -
                                Тип: Таблица<ИсточникДанныхМассив<<ИмяОбъекта>.<ИмяТЧ>>>
                                Имя: <ИмяТЧ>
                                Навигация: ПодгрузкаПриПрокрутке
                                Источник:
                                    Данные: =Объект.<ИмяТЧ>
                                Колонки:
                                    -
                                        Тип: СтандартнаяКолонкаТаблицы<<ИмяОбъекта>.<ИмяТЧ>>
                                        Заголовок: <Заголовок>
                                        ПолеЗначения: <ИмяКолонки>
                                        НастройкиРедактирования:
                                            Тип: НастройкиРедактированияПоляВвода
                                Команды:
                                    Тип: ФрагментКомандногоИнтерфейса
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ДобавитьСтроку
                                КомандыСтроки:
                                    Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<<ИмяОбъекта>.<ИмяТЧ>>>
                                    Элементы:
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВверх
                                        - =Компоненты.<ИмяТЧ>.ПереместитьСтрокуВниз
                                        - =Компоненты.<ИмяТЧ>.Удалить
```

## Правила

- `ШиринаВКолонках` — количество колонок сетки: `Одинарная`, `Двойная`, `Четверная`
- Для Справочника без ТЧ — `ПроизвольныйШаблонФормы`
- Для Документа или объекта с ТЧ — `ШаблонФормыСРазделами`
- `Обязательное: Истина` + `СообщениеОшибка` — для обязательных полей
- Системные поля (`Наименование`, `Код`, `Номер`, `Дата`) — без `НезаполненноеЗначение`, привязывать напрямую
- `=Объект.Наименование` — привязка к Наименованию
- `=Объект.Номер` / `=Объект.Дата` — для документов (поле Номер делать с `Доступность: Ложь` если автонумерация)

## Свойства формы (реактивное состояние)

```yaml
Свойства:
    -
        Имя: <ИмяСвойства>
        Тип: Булево
        ЗначениеПоУмолчанию: Ложь
    -
        Имя: <ИмяСвойства>
        Тип: Строка
        Контекстное: Истина   # передаётся на сервер в @Контекстный методах
```

Добавляй свойства только если пользователь явно просит интерактивное поведение.

## XBSL обработчики

```xbsl
@Обработчик
метод ПередЗаписью()
    // логика перед сохранением
;

@Обработчик
метод ПослеЗаписи()
    // логика после сохранения
;

@НаСервере @ДоступноСКлиента
@Контекстный
метод МойКонтекстныйМетод()
    // выполняется на сервере, доступен с клиента
    // имеет доступ к Объект и Контекстным свойствам
;
```
