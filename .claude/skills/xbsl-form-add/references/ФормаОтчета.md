# Спецификация: ФормаОтчета

## Структура

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОтчета>ФормаОтчета
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаОтчета
    ВключатьВАвтоИнтерфейс: Ложь
    Заголовок: =Отчет.Представление
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        Содержимое:
            Тип: Группа
            Компоновка: Вертикальная
            Содержимое:
                # --- Параметры (если report_params не пуст) ---
                -
                    Тип: Группа
                    Содержимое:
                        -
                            Тип: ПолеВвода<<Тип>>
                            Заголовок: <Представление параметра>
                            Значение: =Отчет.Параметры.<ИмяПараметра>
                # --- Просмотр отчёта (всегда) ---
                -
                    Тип: ПросмотрОтчета
                    Имя: ПросмотрОтчета
                    Отчет: =Отчет
                    РастягиватьПоВертикали: Истина
                    РастягиватьПоГоризонтали: Истина
                    ТолькоЧтение: Истина
Свойства:
    -
        Имя: Отчет
        Тип: <ИмяОтчета>
        ЗначениеПоУмолчанию:
            Тип: <ИмяОтчета>
```

## Вариант без параметров (report_params пуст)

Если `report_params` пуст — убираем внешнюю `Группа` с `Компоновка: Вертикальная` и секцию параметров, оставляем только `ПросмотрОтчета`:

```yaml
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        Содержимое:
            Тип: ПросмотрОтчета
            Имя: ПросмотрОтчета
            Отчет: =Отчет
            РастягиватьПоВертикали: Истина
            РастягиватьПоГоризонтали: Истина
            ТолькоЧтение: Истина
```

## Правила

- `ВключатьВАвтоИнтерфейс: Ложь` — **всегда**
- `Заголовок: =Отчет.Представление` — **всегда** (заголовок берётся из объекта отчёта)
- Параметры — отображаются через `=Отчет.Параметры.<ИмяПараметра>`. Тип ПолеВвода подбирается по типу параметра:
  - `ДатаВремя` → `ПолеВвода<ДатаВремя>`
  - `Дата?` → `ПолеВвода<Дата?>`
  - `<Справочник>.Ссылка?` → `ПолеВвода<<Справочник>.Ссылка?>`
  - `Число` → `ПолеВвода<Число>`
  - `Строка` → `ПолеВвода<Строка>`
- Для ссылочных параметров поле `Заголовок` обязательно
- `ТолькоЧтение: Истина` на ПросмотрОтчета — **всегда** (отчёты только для просмотра)
- Секция `Свойства` — обязательна: содержит экземпляр объекта отчёта, который передаётся в форму

## Обновление объекта Отчет после создания формы (Шаг 6)

Для объекта типа `Отчет` секция `Интерфейс` имеет плоскую структуру (не через `Объект`/`Список`):

```yaml
Интерфейс:
    ВключатьВАвтоИнтерфейс: Истина
    Форма: <ИмяОтчета>ФормаОтчета
```

Добавляется **после `ОбластьВидимости`**, до `ВидИсточникаДанных` / `ПараметрыЗапроса`.
