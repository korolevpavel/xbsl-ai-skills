# Спецификация: ФормаСписка

## Дефолты

- **Имя** — `{ИмяОбъекта}ФормаСписка` (PascalCase)
- **Видимость** — `ВПодсистеме`
- **UUID** — ровно 1 на форму
- **ВключатьВАвтоИнтерфейс** — всегда `Ложь`
- Таблица называется `ОсновнаяТаблица`, компонент — `=Компоненты.ОсновнаяТаблица`
- Динамический список описывается в `Свойства` (не внутри Содержимое)

## Структура YAML

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаСписка
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаСписка
    ВключатьВАвтоИнтерфейс: Ложь
    Заголовок: <Заголовок>
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            - =Обновить
    КомандыСоздания: =Создать
    КомпонентТаблицы: =Компоненты.ОсновнаяТаблица
    Содержимое:
        Тип: ПроизвольныйШаблонФормы

            Тип: Таблица<ДинамическийСписок<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>

                -
                    Тип: СтандартнаяКолонкаТаблицы<СтрокаДинамическогоСписка<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>
                    Имя: <Поле2>
                    Значение: =ДанныеСтроки.Данные.<Поле2>
            КомандыСтроки:
                Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<СтрокаДинамическогоСписка<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>>
```

## Правила

- `ВключатьВАвтоИнтерфейс: Ложь` — обязательно всегда
- `Источник: =Список` — таблица ссылается на свойство `Список`, определённое в `Свойства`
- Колонки: `Значение: =ДанныеСтроки.Данные.<Поле>` — так привязываются данные строки
- `Имя` колонки — произвольное, обычно совпадает с именем поля
- Поля в `Свойства.ЗначениеПоУмолчанию.Поля` — должны совпадать с полями в колонках

### Выбор колонок

Добавляй **все поля объекта** как колонки. Для Справочника первая колонка — всегда `Наименование`, затем остальные поля в порядке из YAML объекта.

## Сортировка

```yaml
            ЗначениеПоУмолчанию:
                ...
                Сортировка:
                    -
                        Поле: <ПолеСортировки>
                        НаправлениеСортировки: ПоУбыванию   # или ПоВозрастанию
```

Документы обычно сортируются по `Дата` убыванию. Справочники — по `Наименование` возрастанию.

## Колонка со статусом (перечисление с цветом)

```yaml
                -
                    Тип: СтандартнаяКолонкаТаблицы<СтрокаДинамическогоСписка<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>
                    Имя: Статус
                    Значение: =ДанныеСтроки.Данные.Статус
                    Вид: Значок
                    Цвет: =ДанныеСтроки.Данные.Статус.ПолучитьЦвет()
```

```yaml
                    -
                        Тип: КомандаСПараметром<Массив<<ИмяОбъекта>.Ссылка>>

                        Изображение: <Иконка>.svg

```
