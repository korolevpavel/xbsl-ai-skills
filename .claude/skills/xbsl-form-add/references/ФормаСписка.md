# Спецификация: ФормаСписка

## Дефолты

- **Имя** — `{ИмяОбъекта}ФормаСписка` (PascalCase)
- **Видимость** — `ВПодсистеме`
- **UUID** — ровно 1 на форму
- **Сортировка** — по умолчанию по убыванию даты (для Документов) или по Наименованию (для Справочников)

## UUID

Нужен: 1 (сам КомпонентИнтерфейса).

## Структура YAML

### Стандартная ФормаСписка (таблица с колонками):

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: <ИмяОбъекта>ФормаСписка
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ФормаСписка
    Заголовок: $ЛокализованныеСтроки.<КлючПредставленияМн>
    КомпонентТаблицы: =Компоненты.ТаблицаСписка
    КомандыСоздания: =Создать
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        Содержимое:
            Тип: Таблица<ДинамическийСписок<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>
            Имя: ТаблицаСписка
            Навигация: ПодгрузкаПриПрокрутке
            ОтображатьНастройки: Истина
            ПрокруткаПоВертикали: Истина
            Источник:
                ИмяТипаДанныхСтроки: ДанныеСтрокиСписка
                ОсновнаяТаблица:
                    Таблица: <ИмяОбъекта>
                Поля:
                    -
                        Тип: ПолеДинамическогоСписка
                        Выражение: <Поле1>
                    -
                        Тип: ПолеДинамическогоСписка
                        Выражение: <Поле2>
                Сортировка:
                    -
                        Поле: <ПолеСортировки>
                        НаправлениеСортировки: ПоУбыванию   # или ПоВозрастанию
            Колонки:
                -
                    Тип: СтандартнаяКолонкаТаблицы<СтрокаДинамическогоСписка<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>
                    Заголовок: $ЛокализованныеСтроки.<КлючЗаголовка>
                    ПолеЗначения: <Поле1>
                -
                    Тип: СтандартнаяКолонкаТаблицы<СтрокаДинамическогоСписка<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>
                    Заголовок: $ЛокализованныеСтроки.<КлючЗаголовка>
                    ПолеЗначения: <Поле2>
```

### ФормаСписка с фильтром и цветом колонки:

```yaml
            Колонки:
                -
                    Тип: СтандартнаяКолонкаТаблицы<СтрокаДинамическогоСписка<<ИмяОбъекта>ФормаСписка.ДанныеСтрокиСписка>>
                    Заголовок: $ЛокализованныеСтроки.Статус
                    ПолеЗначения: Статус
                    Вид: Значок                                    # для перечислений со значком
                    Цвет: =ДанныеСтроки.Данные.Статус.ПолучитьЦвет()
```

### ФормаСписка с действиями над отмеченными строками:

```yaml
            ИспользоватьОтметкиСтрок: Истина
            КомандыНадОтмеченнымиСтроками:
                Тип: ФрагментКомандногоИнтерфейса<КомандаСПараметром<Массив<<ИмяОбъекта>.Ссылка>>>
                Элементы:
                    -
                        Тип: КомандаСПараметром<Массив<<ИмяОбъекта>.Ссылка>>
                        Представление: <ТекстКоманды>
                        Изображение: <Иконка>.svg
                        Обработчик: <ИмяОбработчика>
```

### ФормаСписка с фильтрацией полей:

```yaml
                Поля:
                    -
                        Тип: ПолеДинамическогоСписка
                        Выражение: Статус
                        ОтображатьВНастройкахФильтров: Истина
                        ОтображатьВПростыхФильтрах: Истина
                    -
                        Тип: ПолеДинамическогоСписка
                        Выражение: Номер
                        ОтображатьВНастройкахФильтров: Ложь
                        ОтображатьВПростыхФильтрах: Ложь
```

### ФормаСписка с вычисляемым полем (JOIN):

```yaml
                Поля:
                    -
                        Тип: ПолеДинамическогоСписка
                        Выражение: Владелец.Клиент     # поле через точку
                        Псевдоним: Владелец_Клиент
                        Представление: Клиент
```

## Правила

- `КомпонентТаблицы: =Компоненты.ТаблицаСписка` — обязательно для стандартной ФормаСписка
- `КомандыСоздания: =Создать` — кнопка создания новой записи
- `ИмяТипаДанныхСтроки: ДанныеСтрокиСписка` — имя типа данных строки (фиксировано)
- Колонка для Справочника: первой колонкой всегда `Наименование`
- Колонка для Документа: первые колонки — `Номер`, `Дата`
- Тип колонки всегда: `СтандартнаяКолонкаТаблицы<СтрокаДинамическогоСписка<<ИмяФормы>.ДанныеСтрокиСписка>>`

## Дополнительные команды в тулбаре

```yaml
    ДополнительныеКоманды:
        Тип: ФрагментКомандногоИнтерфейса<Команда>
        Элементы:
            -
                Тип: ОбычнаяКоманда
                Представление: <ТекстКоманды>
                Изображение: <Иконка>.svg
                Доступность: =<Выражение>
                Обработчик: <ИмяОбработчика>
```
