# P1: KPI-карточка (динамический счётчик)

Карточка с числовым значением, обновляемым при загрузке формы.

```yaml
-
    Тип: СтандартнаяКарточка
    Заголовок: <Название счётчика>
    МинимальнаяВысота: 230
    ОбрабатыватьНажатие: Истина
    ПриНажатии: Карточка<Имя>ПриНажатии
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Двойная          # или Одинарная для компактного счётчика
    Картинка:
        Тип: Картинка
        Изображение: Плюс.svg         # или Задачи.svg
        Цвет: =Цвета.Оранжевый        # Оранжевый | Зеленый | Голубой | Красный | Фиолетовый
    Команды:                          # необязательно — кнопка «Создать»
        Тип: ОбычнаяКоманда
        Важность: Обычная
        Видимость: Истина
        Доступность: Истина
        Изображение: Плюс.svg
        Обработчик: Создать<Объект>ПриНажатии
        ОпасностьДействия: Отсутствует
        Представление: Создать <объект>
    Содержимое:
        Тип: Группа
        Компоновка: Горизонтальная
        Содержимое:
            -
                Тип: Группа
                ИнтервалМеждуЭлементамиПоВертикали: Отсутствует
                Компоновка: Вертикальная
                РастягиватьПоГоризонтали: Истина
                Содержимое:
                    -
                        Тип: Надпись
                        Важность: Низкая
                        Значение: "<Метка>"     # строка в кавычках — обязательно!
                    -
                        Тип: Надпись
                        Имя: НадписьЗначение1   # ← Имя обязательно — обновляется из .xbsl
                        МаксимальнаяВысотаВСтроках: 1
                        Шрифт: =СтилевыеШрифты.ЗаголовокОбычный
```

**Правила:**
- Каждая Надпись со значением **обязана** иметь `Имя:` — иначе нельзя обновить из .xbsl
- Одна карточка может содержать несколько пар метка+значение в горизонтальной Группе

**Фрагмент .xbsl:**
```xbsl
метод ЗаполнитьКарточку<Имя>()
    Компоненты.НадписьЗначение1.Значение = Получить<Имя>().Представление()
;

метод Карточка<Имя>ПриНажатии(Источник: Карточка, Событие: СобытиеПриНажатии)
    <Имя>ФормаСписка.Открыть()
;

метод Создать<Объект>ПриНажатии(Команда: ОбычнаяКоманда)
    <Объект>ФормаОбъекта.Открыть(СпособОткрытия = СпособОткрытияФормы.ВДиалоговомОкне)
;

@НаСервере @ДоступноСКлиента
статический метод Получить<Имя>(): Число
    знч Запрос = Запрос{
        ВЫБРАТЬ
            КОЛИЧЕСТВО(*) КАК Количество
        ИЗ
            <ОбъектЗапроса>
    }
    возврат Запрос.Выполнить().Единственный().Количество
;
```
