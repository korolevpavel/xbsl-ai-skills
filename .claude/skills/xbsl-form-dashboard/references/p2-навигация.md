# P2: Навигационная карточка (клик → открыть форму)

```yaml
-
    Тип: СтандартнаяКарточка
    Заголовок: <Название раздела>
    МинимальнаяВысота: 230
    ОбрабатыватьНажатие: Истина
    ПриНажатии: Карточка<Раздел>ПриНажатии
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Двойная
    Картинка:
        Тип: Картинка
        Изображение: Задачи.svg
        Цвет: =Цвета.Голубой
    Содержимое:
        Тип: Надпись
        Значение: "<Описание раздела>"    # строка в кавычках
```

**Обработчик в .xbsl:**
```xbsl
// Открыть форму списка:
метод Карточка<Раздел>ПриНажатии(Источник: Карточка, Событие: СобытиеПриНажатии)
    <Раздел>ФормаСписка.Открыть()
;

// Открыть форму в диалоговом окне:
метод Карточка<Раздел>ПриНажатии(Источник: Карточка, Событие: СобытиеПриНажатии)
    <Раздел>ФормаСписка.Открыть(СпособОткрытия = СпособОткрытияФормы.ВДиалоговомОкне)
;
```

**Важно:** `<Раздел>ФормаСписка` должна иметь `ОбластьВидимости: ВПроекте` (если в другой подсистеме).
