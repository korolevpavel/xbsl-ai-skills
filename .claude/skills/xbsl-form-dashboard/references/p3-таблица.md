# P3: Табличная карточка (данные из массива)

## Структура данных (Структура YAML)

Для `Таблица<ИсточникДанныхМассив<T>>` и `ПроизвольныйСписок<ИсточникДанныхМассив<T>>`
нужна отдельная `Структура` в той же (или доступной без цикл. зависимости) подсистеме:

```yaml
ВидЭлемента: Структура
Ид: <UUID>
Имя: СтрокаМоегоОбъекта
ОбластьВидимости: ВПроекте
Окружение: КлиентИСервер     # ← обязательно!
Поля:
    -
        Имя: Поле1           # ← НЕ писать Ид: внутри полей
        Тип: Строка
    -
        Имя: Поле2
        Тип: Число
    -
        Имя: Фото
        Тип: ДвоичныйОбъект.Ссылка?
```

## Матричная сетка карточек (P3b — аналог «Наши сотрудники»)

```yaml
-
    Тип: СтандартнаяКарточка
    Заголовок: <Название>
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Четверная
    Содержимое:
        Тип: ПроизвольныйСписок<ИсточникДанныхМассив<СтрокаМоегоОбъекта>>
        Имя: СписокМоегоОбъекта
        ИспользованиеПоиска: Ложь
        ОтображатьНастройки: Ложь
        ТипКомпонентаСтроки: КарточкаМоегоОбъекта   # ← отдельный КомпонентИнтерфейса
        КонтейнерСтрок:
            Тип: Группа
            Компоновка: Матричная
            РастягиватьПоГоризонтали: Истина
            НастройкиМатричнойКомпоновки:
                АвтоЗаполнение: ДобавлятьКолонкиИСтроки
                ОписаниеАвтоматическихКолонок:
                    МинимальнаяШирина: 200
                    РастягиватьПоГоризонтали: Истина
                ОписаниеАвтоматическихСтрок:
                    РастягиватьПоВертикали: Истина
```

**Компонент строки (`КарточкаМоегоОбъекта.yaml`):**
```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: КарточкаМоегоОбъекта
ОбластьВидимости: ВПодсистеме
Наследует:
    Тип: ПроизвольнаяСтрокаСписка<СтрокаМоегоОбъекта>
    Содержимое:
        Тип: ПроизвольнаяКарточка
        РастягиватьПоВертикали: Истина
        РастягиватьПоГоризонтали: Истина
        Содержимое:
            Тип: Группа
            Компоновка: Вертикальная
            РастягиватьПоГоризонтали: Истина
            Содержимое:
                -
                    Тип: Картинка
                    Высота: 160
                    Масштабирование: Пропорционально
                    РастягиватьПоГоризонтали: Истина
                    Изображение: =ДанныеСтроки.Фото ?? Ресурс{Аккаунт.svg}.Ссылка
                -
                    Тип: Надпись
                    РастягиватьПоГоризонтали: Истина
                    ВыравниваниеВГруппеПоГоризонтали: Центр
                    Значение: =ДанныеСтроки.Поле1
```

Доступ к данным строки: `=ДанныеСтроки.ИмяПоля` (не `=ДанныеСтроки.Данные.ИмяПоля`).

## Фрагмент .xbsl

```xbsl
метод ЗаполнитьТаблицу<Имя>()
    Компоненты.Таблица<Имя>.Источник.Данные = Получить<Имя>()
;

@НаСервере @ДоступноСКлиента
статический метод Получить<Имя>(): Массив<СтрокаМоегоОбъекта>
    знч Результат = новый Массив<СтрокаМоегоОбъекта>()
    знч Запрос = Запрос{
        ВЫБРАТЬ
            Поле1.ЗаменитьNull() КАК Поле1,    // .ЗаменитьNull() — если поле может быть Null
            Поле2 КАК Поле2
        ИЗ
            МойОбъект
        УПОРЯДОЧИТЬ ПО
            Поле1
    }
    для Строка из Запрос.Выполнить()             // ← без знч, без цикл/конец
        Результат.Добавить(новый СтрокаМоегоОбъекта(Поле1 = Строка.Поле1, Поле2 = Строка.Поле2))
    ;
    возврат Результат
;
```

**Синтаксис цикла:** `для <Переменная> из <Коллекция> ... ;` (без `знч`, `цикл`, `конец`).
**Инициализация структуры:** `новый Тип(Поле1 = Значение1, Поле2 = Значение2)`.
**Nullable поля в запросе:** используй `.ЗаменитьNull()` для Строка?, или `.ЗаменитьNull(0)` для Число?.
