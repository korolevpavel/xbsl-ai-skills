# P4: Баннер (картинка + текст + кнопка)

```yaml
-
    Тип: СтандартнаяКарточка
    ВидОтображения: Баннер
    Заголовок: <Заголовок баннера>
    МинимальнаяВысота: 230
    РасположениеИзображенияВБаннере: ПослеСодержимого   # ДоСодержимого | ПослеСодержимого
    РастягиватьИзображениеВБаннере: Истина
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Четверная                          # баннер обычно на всю ширину
    Фон:
        Тип: АбсолютныйЦвет
        Значение: RGB(5955FF)                           # или =Цвета.Стилевые.ФонПервичный
    ЦветЗаголовка:
        Тип: АбсолютныйЦвет
        Значение: RGB(FFFFFF)
    ЦветСодержимого:
        Тип: АбсолютныйЦвет
        Значение: RGB(FFFFFF)
    Картинка:
        Тип: Картинка
        АльтернативныйТекст: <Описание изображения>
        Изображение: <Файл.png>                         # файл в Ресурсы/ подсистемы
        Масштабирование: Пропорционально
    Команды:
        Тип: ГруппаКомандногоИнтерфейса
        Элементы:
            -
                Тип: ОбычнаяКоманда
                Важность: Высокая
                Видимость: Истина
                Доступность: Истина
                ОпасностьДействия: Отсутствует
                Представление: <Текст кнопки>
                Обработчик: <Обработчик>ПриНажатии
    Содержимое: "<Текст описания>"                      # строка в кавычках или Надпись
    ШрифтЗаголовка:
        Тип: АбсолютныйШрифт
        Полужирный: Истина
        Размер: 36
    ШрифтСодержимого: =СтилевыеШрифты.ОченьКрупный
```

**Правила:**
- `ВидОтображения: Баннер` — обязательно
- Фон: предпочитать `=Цвета.Стилевые.ФонПервичный` для брендовых цветов
  или `RGB(...)` для кастомного
- Картинка — файл в `<Подсистема>/Ресурсы/` (не платформенные SVG)
- `Содержимое:` баннера — строковый литерал в кавычках или вложенная `Надпись`

**Простой баннер (без кнопки, без кастомных цветов):**
```yaml
-
    Тип: СтандартнаяКарточка
    ВидОтображения: Баннер
    Заголовок: <Заголовок>
    РастягиватьИзображениеВБаннере: Истина
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Двойная
    Картинка:
        Тип: Картинка
        Изображение: Плюс.svg
    Содержимое: "<Описание>"
```
