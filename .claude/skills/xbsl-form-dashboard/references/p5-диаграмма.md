# P5: Диаграмма

## Доступные типы диаграмм (проверено на реальных проектах)

| Тип в YAML | Описание | Тип серии |
|---|---|---|
| `КруговаяДиаграмма<T>` | Круговая / пончик | `КруговаяСерияДиаграммы` |
| `XYДиаграмма<T, S>` | Столбчатая / линейная | `СтолбчатаяСерияДиаграммы` или `ЛинейнаяСерияДиаграммы` |
| `ДиаграммаСпидометр<T>` | Спидометр | `СерияДиаграммыСпидометр` |
| `ВоронкообразнаяДиаграмма<T>` | Воронка | `ВоронкообразнаяСерияДиаграммы` |

> **Устаревшие названия** (не существуют): `ДиаграммаКруговая`, `ДиаграммаВоронкаПродаж`, `ДиаграммаЛинейная` — не использовать.

---

## Ключевое правило: серии создаются из XBSL, не из YAML

**Серию можно объявить в YAML** (тип + имя), но без настройки `ПоляДанных` из XBSL диаграмма будет пустой.

**Правильный подход**: серии, оси и ПоляДанных — всё создавать из XBSL в методе `НастроитьДиаграммы()`, который вызывается ДО `ЗаполнитьДиаграммы()`.

```yaml
# ❌ Не работает — серия без ПоляДанных даёт пустую диаграмму:
Серии:
    -
        Тип: КруговаяСерияДиаграммы
        Имя: СерияОстатки

# ✅ Правильно — в YAML только компонент без Серии:
Тип: КруговаяДиаграмма<СтрокаОстатков>
Имя: ДиаграммаОстатки
Высота: 200
```

```yaml
# ❌ Ошибка компиляции — ПоляДанных нельзя задать в YAML:
ПоляДанных:
    Ключ: "Наименование"    # Значение типа ПолеДанныхДиаграммы не может быть описано в виде литерала
    Значение: "Количество"
```

## КруговаяДиаграмма

### YAML
```yaml
Тип: КруговаяДиаграмма<СтрокаОстатков>
Имя: ДиаграммаОстатки
Высота: 200
```

### XBSL — настройка серии и легенды
```xbsl
метод НастроитьДиаграммы()
    знч Диаграмма = Компоненты.ДиаграммаОстатки
    знч Серия = новый КруговаяСерияДиаграммы()
    Серия.ПоляДанных.Ключ.ПолеЗначения = "Наименование"
    Серия.ПоляДанных.Ключ.ОтображаемоеЗначение = "Наименование"
    Серия.ПоляДанных.Значение.ПолеЗначения = "Количество"
    Серия.ПоляДанных.Значение.ОтображаемоеЗначение = "Количество"
    Диаграмма.Серии.Добавить(Серия)
    Диаграмма.Легенда = новый ЛегендаДиаграммы()
    Диаграмма.Легенда.Расположение = РасположениеЛегендыДиаграммы.Конец
;

метод ЗаполнитьДиаграммы()
    Компоненты.ДиаграммаОстатки.Источник.Данные = ПолучитьОстатки()
;
```

### Структура данных
```yaml
# Отдельный файл СтрокаОстатков.yaml (ОбластьВидимости: ВПроекте, Окружение: КлиентИСервер)
Поля:
    - Имя: Наименование
      Тип: Строка
    - Имя: Количество
      Тип: Число
```

## ВоронкообразнаяДиаграмма

Имеет дополнительные свойства для отображения меток и долей.

### YAML
```yaml
Тип: ВоронкообразнаяДиаграмма<СтрокаЭтапа>
Имя: ДиаграммаВоронка
Высота: 200
```

### XBSL — настройка серии
```xbsl
метод НастроитьДиаграммы()
    знч Диаграмма = Компоненты.ДиаграммаВоронка
    Диаграмма.ОтображатьВнутренниеМаркеры = Истина
    Диаграмма.ОтображатьВнешниеМаркеры = Истина
    Диаграмма.ОтображатьОтносительныеДоли = Истина
    Диаграмма.ОтображатьАбсолютныеДоли = Истина
    знч Серия = новый ВоронкообразнаяСерияДиаграммы()
    Серия.ПоляДанных.Ключ.ПолеЗначения = "Этап"
    Серия.ПоляДанных.Ключ.ОтображаемоеЗначение = "Этап"
    Серия.ПоляДанных.Значение.ПолеЗначения = "Количество"
    Серия.ПоляДанных.Значение.ОтображаемоеЗначение = "Количество"
    Серия.ПоляДанных.Цвет = "Цвет"    // опционально
    Диаграмма.Серии.Добавить(Серия)
;

метод ЗаполнитьДиаграммы()
    Компоненты.ДиаграммаВоронка.Источник.Данные = ПолучитьВоронку()
;
```

### Структура данных
```yaml
# СтрокаЭтапа.yaml (ОбластьВидимости: ВПроекте, Окружение: КлиентИСервер)
Поля:
    - Имя: Этап
      Тип: Строка
    - Имя: Количество
      Тип: Число
```

### XBSL — пример заполнения (Заказы → Реализации)
```xbsl
@НаСервере @ДоступноСКлиента
статический метод ПолучитьВоронку(): Массив<СтрокаЭтапа>
    знч Результат = новый Массив<СтрокаЭтапа>()
    знч ЗапросЗаказов = Запрос{ ВЫБРАТЬ КОЛИЧЕСТВО(*) КАК Количество ИЗ ЗаказКлиента }
    знч ЗапросРеализаций = Запрос{ ВЫБРАТЬ КОЛИЧЕСТВО(*) КАК Количество ИЗ Реализация }
    Результат.Добавить(новый СтрокаЭтапа(Этап = "Заказы", Количество = ЗапросЗаказов.Выполнить().Единственный().Количество))
    Результат.Добавить(новый СтрокаЭтапа(Этап = "Реализации", Количество = ЗапросРеализаций.Выполнить().Единственный().Количество))
    возврат Результат
;
```

## XYДиаграмма (столбчатая)

Требует явного создания осей X и Y из XBSL. Без осей — ошибка "Не задана ни одна ось X".

### YAML
```yaml
Тип: XYДиаграмма<СтрокаДанных, СтолбчатаяСерияДиаграммы>
Имя: ДиаграммаЗаказы
Высота: 200
```

### XBSL — создание осей и серии
```xbsl
метод НастроитьДиаграммы()
    знч Диаграмма = Компоненты.ДиаграммаЗаказы
    // Ось X (категориальная — строковые метки)
    знч ОсьX = новый КатегориальнаяОсьДиаграммы()
    ОсьX.ПолеДанных.ПолеЗначения = "Категория"
    ОсьX.ПолеДанных.ОтображаемоеЗначение = "Категория"
    Диаграмма.ОсиX.Добавить(ОсьX)
    // Ось Y (числовая)
    знч ОсьY = новый ЧисловаяОсьДиаграммы()
    Диаграмма.ОсиY.Добавить(ОсьY)
    // Серия
    знч Серия = новый СтолбчатаяСерияДиаграммы()
    Серия.ПоляДанных.Ключ.ПолеЗначения = "Категория"
    Серия.ПоляДанных.Ключ.ОтображаемоеЗначение = "Категория"
    Серия.ПоляДанных.Значение.ПолеЗначения = "Сумма"
    Серия.Ось = ОсьY
    Диаграмма.Серии.Добавить(Серия)
;

метод ЗаполнитьДиаграммы()
    Компоненты.ДиаграммаЗаказы.Источник.Данные = ПолучитьДанные()
;
```

> **Важно:** `ФОРМАТ(Дата, "...")` в запросах не поддерживается. Для группировки по дате используй `НАЧАЛОПЕРИОДА()` или `МЕСЯЦ()`.
>
> Для полей через `.` (Зк.Склад.Наименование) — обязателен алиас таблицы `КАК Зк`. Без алиаса — ошибка `Поле "ЗаказКлиента" не найдено`.
>
> Поля-ссылки (Склад, Организация) возвращают UUID если не взять `.Наименование`. Группируй по `Зк.Склад.Наименование КАК Категория`.
>
> `ЕСТЬNULL()` не поддерживается. Для защиты от null используй `.ЗаменитьNull()` прямо в SQL-выражении: `СУММА(Сумма).ЗаменитьNull(0) КАК Сумма`.

## ДиаграммаСпидометр

**Требует**: пользовательскую структуру с зонами шкалы + `Источник.Данные`. Просто `Значение` без зон даёт "Нет данных для отображения".

### Шаг 1 — Структура зон (в .xbsl файле формы)
```xbsl
@ВПодсистеме
структура ЗонаСпидометра
    обз знч Обозначение: Строка
    обз знч Значение: Число    // верхняя граница зоны
    обз знч Цвет: Цвет
;
```

### Шаг 2 — YAML (тип ссылается на структуру формы)
```yaml
Тип: ДиаграммаСпидометр<МояФорма.ЗонаСпидометра>
Имя: ДиаграммаСпидометр
Высота: 200
МинимальноеЗначение: 0
```
Где `МояФорма` — имя файла формы (например `ГлавнаяСтраница`).

### Шаг 3 — XBSL
```xbsl
метод НастроитьДиаграммы()
    знч Диаграмма = Компоненты.ДиаграммаСпидометр
    // Зоны определяют шкалу (верхние границы)
    знч Зоны = новый Массив<ЗонаСпидометра>()
    Зоны.Добавить(новый ЗонаСпидометра(Обозначение = "Низкий", Значение = 5000, Цвет = Цвета.Зеленый))
    Зоны.Добавить(новый ЗонаСпидометра(Обозначение = "Средний", Значение = 15000, Цвет = Цвета.Желтый))
    Зоны.Добавить(новый ЗонаСпидометра(Обозначение = "Высокий", Значение = 50000, Цвет = Цвета.Красный))
    Диаграмма.Источник.Данные = Зоны
    знч Серия = новый СерияДиаграммыСпидометр()
    Серия.ПоляДанных.Ключ.ПолеЗначения = "Обозначение"
    Серия.ПоляДанных.Ключ.ОтображаемоеЗначение = "Обозначение"
    Серия.ПоляДанных.Значение.ПолеЗначения = "Значение"
    Серия.ПоляДанных.Значение.ОтображаемоеЗначение = "Значение"
    Серия.ПоляДанных.Цвет = "Цвет"
    Диаграмма.Серии.Добавить(Серия)
;

метод ЗаполнитьДиаграммы()
    знч Значение = ПолучитьЗначение()
    Компоненты.ДиаграммаСпидометр.Значение = Значение
    Компоненты.ДиаграммаСпидометр.ОтображаемоеЗначение = Значение.Представление() + " руб."
;
```

## Шаблон: бенто-сетка с 3 диаграммами

```yaml
-
    Тип: СтандартнаяКарточка
    Заголовок: "Аналитика"
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Четверная
    Содержимое:
        Тип: Группа
        Компоновка: Бенто
        НастройкиКомпоновкиБенто:
            РазрешитьИзменениеПорядка: Ложь
        Содержимое:
            -
                Тип: СтандартнаяКарточка
                Заголовок: "Круговая"
                ШиринаВКолонках: Двойная
                РастягиватьПоВертикали: Истина
                Содержимое:
                    Тип: КруговаяДиаграмма<СтрокаОстатков>
                    Имя: ДиаграммаКруговая
                    Высота: 200
            -
                Тип: СтандартнаяКарточка
                Заголовок: "Столбчатая"
                ШиринаВКолонках: Двойная
                РастягиватьПоВертикали: Истина
                Содержимое:
                    Тип: XYДиаграмма<СтрокаДанных, СтолбчатаяСерияДиаграммы>
                    Имя: ДиаграммаСтолбчатая
                    Высота: 200
            -
                Тип: СтандартнаяКарточка
                Заголовок: "Спидометр"
                ШиринаВКолонках: Двойная
                РастягиватьПоВертикали: Истина
                Содержимое:
                    Тип: ДиаграммаСпидометр<ГлавнаяСтраница.ЗонаСпидометра>
                    Имя: ДиаграммаСпидометр
                    Высота: 200
                    МинимальноеЗначение: 0
```

### XBSL (полный шаблон)
```xbsl
// Структура зон для спидометра — в начале .xbsl файла
@ВПодсистеме
структура ЗонаСпидометра
    обз знч Обозначение: Строка
    обз знч Значение: Число
    обз знч Цвет: Цвет
;

// Вызывать НастроитьДиаграммы() ДО ЗаполнитьДиаграммы()
метод НастроитьДиаграммы()
    // Круговая
    знч Д1 = Компоненты.ДиаграммаКруговая
    знч С1 = новый КруговаяСерияДиаграммы()
    С1.ПоляДанных.Ключ.ПолеЗначения = "Наименование"
    С1.ПоляДанных.Ключ.ОтображаемоеЗначение = "Наименование"
    С1.ПоляДанных.Значение.ПолеЗначения = "Количество"
    С1.ПоляДанных.Значение.ОтображаемоеЗначение = "Количество"
    Д1.Серии.Добавить(С1)
    Д1.Легенда = новый ЛегендаДиаграммы()
    Д1.Легенда.Расположение = РасположениеЛегендыДиаграммы.Конец
    // XY столбчатая
    знч Д2 = Компоненты.ДиаграммаСтолбчатая
    знч ОсьX = новый КатегориальнаяОсьДиаграммы()
    ОсьX.ПолеДанных.ПолеЗначения = "Категория"
    ОсьX.ПолеДанных.ОтображаемоеЗначение = "Категория"
    Д2.ОсиX.Добавить(ОсьX)
    знч ОсьY = новый ЧисловаяОсьДиаграммы()
    Д2.ОсиY.Добавить(ОсьY)
    знч С2 = новый СтолбчатаяСерияДиаграммы()
    С2.ПоляДанных.Ключ.ПолеЗначения = "Категория"
    С2.ПоляДанных.Ключ.ОтображаемоеЗначение = "Категория"
    С2.ПоляДанных.Значение.ПолеЗначения = "Сумма"
    С2.Ось = ОсьY
    Д2.Серии.Добавить(С2)
    // Спидометр
    знч Д3 = Компоненты.ДиаграммаСпидометр
    знч Зоны = новый Массив<ЗонаСпидометра>()
    Зоны.Добавить(новый ЗонаСпидометра(Обозначение = "Низкий", Значение = 5000, Цвет = Цвета.Зеленый))
    Зоны.Добавить(новый ЗонаСпидометра(Обозначение = "Средний", Значение = 15000, Цвет = Цвета.Желтый))
    Зоны.Добавить(новый ЗонаСпидометра(Обозначение = "Высокий", Значение = 50000, Цвет = Цвета.Красный))
    Д3.Источник.Данные = Зоны
    знч С3 = новый СерияДиаграммыСпидометр()
    С3.ПоляДанных.Ключ.ПолеЗначения = "Обозначение"
    С3.ПоляДанных.Ключ.ОтображаемоеЗначение = "Обозначение"
    С3.ПоляДанных.Значение.ПолеЗначения = "Значение"
    С3.ПоляДанных.Значение.ОтображаемоеЗначение = "Значение"
    С3.ПоляДанных.Цвет = "Цвет"
    Д3.Серии.Добавить(С3)
;

метод ЗаполнитьДиаграммы()
    Компоненты.ДиаграммаКруговая.Источник.Данные = ПолучитьОстатки()
    Компоненты.ДиаграммаСтолбчатая.Источник.Данные = ПолучитьДанные()
    знч Значение = ПолучитьЧисло()
    Компоненты.ДиаграммаСпидометр.Значение = Значение
    Компоненты.ДиаграммаСпидометр.ОтображаемоеЗначение = Значение.Представление()
;
```
