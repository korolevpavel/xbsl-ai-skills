# P5: Диаграмма

```yaml
-
    Тип: СтандартнаяКарточка
    Заголовок: <Название диаграммы>
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Двойная
    Содержимое:
        Тип: Группа
        Компоновка: Вертикальная
        ОтступПоГоризонтали: Двойной
        РастягиватьПоГоризонтали: Истина
        Содержимое:
            -
                Тип: <ТипДиаграммы>
                Имя: <ИмяДиаграммы>
```

**Доступные типы диаграмм:**
- `ДиаграммаВоронкаПродаж` — воронкообразная
- `ДиаграммаКруговая` — круговая
- `ДиаграммаЛинейная` — линейная
- `ДиаграммаСтековая` — стековая
- `ДиаграммаЛепестковая` — лепестковая
- `ДиаграммаСпидометр` — спидометр
- `ДиаграммаВоронкообразная` — воронкообразная (альтернативный вид)
- `ДиаграммаПузырьковая` — пузырьковая
- `ДиаграммаГанта` — диаграмма Ганта

**Обновление из .xbsl:**
```xbsl
метод ОбновитьДиаграммы()
    Компоненты.<ИмяДиаграммы>.ОбновитьДиаграмму()
;
```

Для круговой и воронки нужно заполнить `Источник.Данные` + настроить `Серии`:
```xbsl
@НаСервере @ДоступноСКлиента
статический метод ПолучитьДанные(): <ТипДанных>
    // Возвращает данные для диаграммы
    // Пример: Соответствие<Ключ, Число> для круговой
;
```

Подробные примеры настройки серий — в `tools/vendor/CRM/Пресейл/ДиаграммаПродажиПоНаправлениям.xbsl`.
