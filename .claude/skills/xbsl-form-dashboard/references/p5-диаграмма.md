# P5: Диаграмма

## Доступные типы диаграмм (проверено на реальных проектах)

| Тип в YAML | Описание | Тип серии |
|---|---|---|
| `КруговаяДиаграмма<T>` | Круговая / пончик | `КруговаяСерияДиаграммы` |
| `ВоронкообразнаяДиаграмма<T>` | Воронка | `ВоронкообразнаяСерияДиаграммы` |
| `XYДиаграмма<T, S>` | Столбчатая / линейная | `СтолбчатаяСерияДиаграммы` или `ЛинейнаяСерияДиаграммы` |
| `ДиаграммаСпидометр<T>` | Спидометр | не нужен, см. ниже |

> **Устаревшие названия** (не существуют): `ДиаграммаКруговая`, `ДиаграммаВоронкаПродаж`, `ДиаграммаЛинейная` — не использовать.

---

## Базовая структура YAML

```yaml
-
    Тип: СтандартнаяКарточка
    Заголовок: "Название диаграммы"
    ШиринаВКолонках: Двойная
    РастягиватьПоВертикали: Истина
    Содержимое:
        Тип: КруговаяДиаграмма<СтрокаОстатков>
        Имя: ДиаграммаОстатки
        Высота: 200
        Серии:
            -
                Тип: КруговаяСерияДиаграммы
                Имя: СерияОстатки
```

**Критично:** `ПоляДанных` (Ключ/Значение серии) **нельзя** задавать в YAML:
```yaml
# ❌ Ошибка компиляции:
ПоляДанных:
    Ключ: "Наименование"    # Значение типа ПолеДанныхДиаграммы не может быть описано в виде литерала
    Значение: "Количество"
```
Маппинг полей настраивается только через XBSL-код (см. ниже).

---

## КруговаяДиаграмма

```yaml
Тип: КруговаяДиаграмма<СтрокаМоихДанных>
Имя: ДиаграммаОстатки
Высота: 200
Серии:
    -
        Тип: КруговаяСерияДиаграммы
        Имя: СерияОстатки
```

XBSL — загрузка данных:
```xbsl
Компоненты.ДиаграммаОстатки.Источник.Данные = ПолучитьОстатки()
```

Структура данных (Структура с ОбластьВидимости: ВПроекте, Окружение: КлиентИСервер):
```yaml
Поля:
    - Имя: Наименование
      Тип: Строка
    - Имя: Количество
      Тип: Число
```

## ВоронкообразнаяДиаграмма

```yaml
Тип: ВоронкообразнаяДиаграмма<СтрокаЭтапаПродаж>
Имя: ДиаграммаВоронка
Высота: 200
Серии:
    -
        Тип: ВоронкообразнаяСерияДиаграммы
        Имя: СерияВоронка
```

XBSL — заполнение воронки (пример Заказы → Реализации):
```xbsl
@НаСервере @ДоступноСКлиента
статический метод ПолучитьВоронкуПродаж(): Массив<СтрокаЭтапаПродаж>
    знч Результат = новый Массив<СтрокаЭтапаПродаж>()
    знч ЗапросЗаказов = Запрос{ ВЫБРАТЬ КОЛИЧЕСТВО(*) КАК Количество ИЗ ЗаказКлиента }
    знч ЗапросРеализаций = Запрос{ ВЫБРАТЬ КОЛИЧЕСТВО(*) КАК Количество ИЗ Реализация }
    Результат.Добавить(новый СтрокаЭтапаПродаж(Этап = "Заказы", Количество = ЗапросЗаказов.Выполнить().Единственный().Количество))
    Результат.Добавить(новый СтрокаЭтапаПродаж(Этап = "Реализации", Количество = ЗапросРеализаций.Выполнить().Единственный().Количество))
    возврат Результат
;
```

## XYДиаграмма (столбчатая)

```yaml
Тип: XYДиаграмма<СтрокаМоихДанных, СтолбчатаяСерияДиаграммы>
Имя: ДиаграммаЗаказы
Высота: 200
Серии:
    -
        Тип: СтолбчатаяСерияДиаграммы
        Имя: СерияЗаказы
```

Запрос с алиасом (обязателен для полей через точку):
```xbsl
знч Запрос = Запрос{
    ВЫБРАТЬ
        Зк.Организация КАК Категория,
        СУММА(Зк.Сумма) КАК Сумма
    ИЗ
        ЗаказКлиента КАК Зк
    СГРУППИРОВАТЬ ПО
        Зк.Организация
    УПОРЯДОЧИТЬ ПО
        Сумма УБЫВ
}
```

> **Важно:** `ФОРМАТ(Дата, "...")` в запросах не поддерживается. Для группировки по дате используй `НАЧАЛОПЕРИОДА()` или `МЕСЯЦ()`.
>
> Для полей через `.` (Зк.Контрагент.Наименование) — обязателен алиас таблицы `КАК Зк`. Без алиаса — ошибка `Поле "ЗаказКлиента" не найдено`.

## ДиаграммаСпидометр

Простейший тип — не нужен `Источник`. Значение задаётся прямым свойством.

```yaml
Тип: ДиаграммаСпидометр<Число>
Имя: ДиаграммаЗадач
Высота: 200
Значение: 0
```

XBSL — обновление значения:
```xbsl
Компоненты.ДиаграммаЗадач.Значение = ПолучитьКоличествоЗадач()
```

## Бенто-сетка с 4 диаграммами (рабочий шаблон)

```yaml
-
    Тип: СтандартнаяКарточка
    Заголовок: "Аналитика"
    РастягиватьПоГоризонтали: Ложь
    ШиринаВКолонках: Четверная
    Содержимое:
        Тип: Группа
        Компоновка: Бенто
        НастройкиКомпоновкиБенто:
            РазрешитьИзменениеПорядка: Ложь
        Содержимое:
            -
                Тип: СтандартнаяКарточка
                Заголовок: "Круговая"
                ШиринаВКолонках: Двойная
                РастягиватьПоВертикали: Истина
                Содержимое:
                    Тип: КруговаяДиаграмма<СтрокаОстатков>
                    Имя: ДиаграммаКруговая
                    Высота: 200
                    Серии:
                        -
                            Тип: КруговаяСерияДиаграммы
                            Имя: СерияОстатки
            -
                Тип: СтандартнаяКарточка
                Заголовок: "Воронка"
                ШиринаВКолонках: Двойная
                РастягиватьПоВертикали: Истина
                Содержимое:
                    Тип: ВоронкообразнаяДиаграмма<СтрокаЭтапа>
                    Имя: ДиаграммаВоронка
                    Высота: 200
                    Серии:
                        -
                            Тип: ВоронкообразнаяСерияДиаграммы
                            Имя: СерияВоронка
            -
                Тип: СтандартнаяКарточка
                Заголовок: "Столбчатая"
                ШиринаВКолонках: Двойная
                РастягиватьПоВертикали: Истина
                Содержимое:
                    Тип: XYДиаграмма<СтрокаДанных, СтолбчатаяСерияДиаграммы>
                    Имя: ДиаграммаСтолбчатая
                    Высота: 200
                    Серии:
                        -
                            Тип: СтолбчатаяСерияДиаграммы
                            Имя: СерияДанных
            -
                Тип: СтандартнаяКарточка
                Заголовок: "Спидометр"
                ШиринаВКолонках: Двойная
                РастягиватьПоВертикали: Истина
                Содержимое:
                    Тип: ДиаграммаСпидометр<Число>
                    Имя: ДиаграммаСпидометр
                    Высота: 200
                    Значение: 0
```

XBSL (обновление всех диаграмм):
```xbsl
метод ЗаполнитьДиаграммы()
    Компоненты.ДиаграммаКруговая.Источник.Данные = ПолучитьДанныеОстатков()
    Компоненты.ДиаграммаВоронка.Источник.Данные = ПолучитьВоронку()
    Компоненты.ДиаграммаСтолбчатая.Источник.Данные = ПолучитьДанные()
    Компоненты.ДиаграммаСпидометр.Значение = ПолучитьЧисло()
;
```
