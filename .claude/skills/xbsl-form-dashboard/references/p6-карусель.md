# P6: Карусель

Прокручиваемый горизонтальный ряд элементов.

```yaml
-
    Тип: Группа
    Компоновка: Карусель
    НастройкиКарусели:
        ИнтервалВоспроизведения: 3с    # автоскролл; убрать если не нужен
        ЦиклическаяНавигация: Истина
        ВидОтображенияКнопок: ВПанели  # Авто | ВПанели — стрелки навигации
    Содержимое:
        -
            Тип: СтандартнаяКарточка
            РастягиватьПоВертикали: Истина
            Фон: =Цвета.Стилевые.ФонВторичный
            ШиринаВКолонках: Одинарная
            Содержимое:
                Тип: Группа
                Компоновка: Вертикальная
                Содержимое:
                    -
                        Тип: Надпись
                        Значение: "<Текст>"
                        МаксимальнаяВысотаВСтроках: 5
                        РастягиватьПоГоризонтали: Истина
                        Шрифт: =СтилевыеШрифты.ОченьКрупный
                    -
                        Тип: Надпись
                        Заголовок: <Подпись (например, должность)>
                        Значение: "<Имя>"
                        Изображение: <Аватар.png>    # миниатюра
        # ... повторить для каждого элемента
```

**Карусель из картинок (логотипы, фото):**
```yaml
-
    Тип: Группа
    Компоновка: Карусель
    НастройкиКарусели:
        ИнтервалВоспроизведения: 3с
        ЦиклическаяНавигация: Истина
    Содержимое:
        -
            Тип: Картинка
            Изображение: <Файл1.png>
        -
            Тип: Картинка
            Изображение: <Файл2.png>
```

**Карусель с динамическими данными (через ПроизвольныйСписок):**

Когда данные загружаются динамически (массив из .xbsl), карусель задаётся через `КонтейнерСтрок`.
Нужен отдельный `КомпонентИнтерфейса` типа `ПроизвольнаяСтрокаСписка<ТипСтроки>` для отображения каждого элемента.

```yaml
# В СтандартнаяКарточка → Содержимое:
Тип: ПроизвольныйСписок<ИсточникДанныхМассив<ТипСтроки>>
Имя: <ИмяСписка>
ИспользованиеПоиска: Ложь
ОтображатьНастройки: Ложь
ТипКомпонентаСтроки: <ИмяКомпонентаСтроки>
КонтейнерСтрок:
    Тип: Группа
    Компоновка: Карусель
    РастягиватьПоГоризонтали: Истина
    НастройкиКарусели:
        ЦиклическаяНавигация: Истина
        ВидОтображенияКнопок: ВПанели
```

В .xbsl заполнение данными — стандартное:
```
метод ЗаполнитьКарусель()
    Компоненты.<ИмяСписка>.Источник.Данные = ПолучитьДанные()
;
```

> **Важно:** если список ранее использовал `Компоновка: Матричная`, для перехода на карусель достаточно заменить `Компоновка` и `НастройкиМатричнойКомпоновки` → `НастройкиКарусели`. Данные и .xbsl-код менять не нужно.
