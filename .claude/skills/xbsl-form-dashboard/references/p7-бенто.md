# P7: Бенто (плиточная сетка)

Плиточная сетка с карточками разного размера. Хорошо для категорий, преимуществ, разделов.

```yaml
-
    Тип: Группа
    Заголовок: <Подзаголовок секции>  # необязательно
    Компоновка: Бенто
    НастройкиКомпоновкиБенто:
        РазрешитьИзменениеПорядка: Ложь
    Содержимое:
        -
            Тип: СтандартнаяКарточка
            Заголовок: <Название>
            ШиринаВКолонках: Двойная    # увеличенная карточка
            ОбрабатыватьНажатие: Истина
            ПриНажатии: Карточка<Раздел>ПриНажатии
            Содержимое: "<Описание>"    # необязательно, строка в кавычках
        -
            Тип: СтандартнаяКарточка
            Заголовок: <Название>
            ШиринаВКолонках: Одинарная  # стандартная карточка
            ОбрабатыватьНажатие: Истина
            ПриНажатии: Карточка<Раздел2>ПриНажатии
        # ... повторить
```

**Правила:**
- Смешивать `ШиринаВКолонках: Одинарная` и `Двойная` для разного визуального веса
- `НастройкиКомпоновкиБенто.РазрешитьИзменениеПорядка: Ложь` — фиксированный порядок
- `ОбрабатыватьНажатие: Истина` + `ПриНажатии:` — обработчик в .xbsl

**Бенто для статичного списка (без кликов, без .xbsl):**
```yaml
-
    Тип: Группа
    Компоновка: Бенто
    Содержимое:
        -
            Тип: СтандартнаяКарточка
            ШиринаВКолонках: Одинарная
            Заголовок: <Название>
            Содержимое: "<Текст>"
            РастягиватьПоВертикали: Истина
        -
            Тип: СтандартнаяКарточка
            ШиринаВКолонках: Одинарная
            Заголовок: <Название>
            Содержимое: "<Текст>"
            РастягиватьПоВертикали: Истина
```

**Обработчики навигации в .xbsl (если ОбрабатыватьНажатие: Истина):**
```xbsl
метод Карточка<Раздел>ПриНажатии(Источник: Карточка, Событие: СобытиеПриНажатии)
    <Раздел>ФормаСписка.Открыть()
;
```
