# P8: Поисковая строка

Поле ввода с кнопкой-иконкой для запуска поиска.

```yaml
-
    Тип: ПолеВвода<Строка>
    ЗамещающийТекст: "<Подсказка, например: Услуга или специалист>"
    РастягиватьПоГоризонтали: Истина
    Команды:
        Тип: ОбычнаяКоманда
        Изображение: Лупа.svg
        Представление: Поиск
        Обработчик: ВыполнитьПоискПриНажатии
```

**Обработчик в .xbsl:**
```xbsl
метод ВыполнитьПоискПриНажатии(Команда: ОбычнаяКоманда)
    знч Текст = Компоненты.<ИмяПоля>.Значение
    // логика поиска
;
```

**Примечание:** `Лупа.svg` — встроенная платформенная иконка, всегда доступна.
