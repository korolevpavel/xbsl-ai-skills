# P9: Статичные карточки (без сервера)

Когда вся информация известна заранее и не нужны серверные данные.
В этом случае `.xbsl` файл **не создаётся**.

Типичные применения: преимущества продукта, возможности платформы, FAQ-блок, описание разделов.

```yaml
-
    Тип: Группа
    Компоновка: Бенто
    Содержимое:
        -
            Тип: СтандартнаяКарточка
            ШиринаВКолонках: Одинарная
            Заголовок: <Преимущество 1>
            Содержимое: "<Описание>"
            РастягиватьПоВертикали: Истина
        -
            Тип: СтандартнаяКарточка
            ШиринаВКолонках: Одинарная
            Заголовок: <Преимущество 2>
            Содержимое: "<Описание>"
            РастягиватьПоВертикали: Истина
        -
            Тип: СтандартнаяКарточка
            ШиринаВКолонках: Двойная
            Заголовок: <Ключевая возможность>
            Содержимое: "<Подробное описание>"
            РастягиватьПоВертикали: Истина
```

**Правило:** `Содержимое:` — строка в кавычках (`"Текст"`), не голый текст без кавычек.

**Когда P9 недостаточно и нужен .xbsl:**
- Если карточки кликабельны (`ОбрабатыватьНажатие: Истина`) → добавить обработчики → появляется .xbsl
- Если содержимое карточек берётся из базы данных → использовать P3 (таблица)
