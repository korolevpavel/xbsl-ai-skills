# Шаблон .xbsl файла для ГлавнойСтраницы

Создавать только если есть динамические данные (P1, P2, P3, P5) или обработчики навигации.
Для чисто статических страниц (P4, P6, P7, P9 без кликов) — .xbsl **не нужен**.

```xbsl
импорт <Подсистема1>
импорт <Подсистема2>

@Обработчик
метод ПослеСоздания()
    ЗаполнитьКарточки()

    // Подписаться на события обновления (если нужно):
    // <СобытиеОбновления>.ПодключитьОбработчик(() -> ЗаполнитьКарточки())
;

метод ЗаполнитьКарточки()
    ЗаполнитьКарточку<Имя1>()
    ЗаполнитьКарточку<Имя2>()
    // ОбновитьДиаграммы()
;

// P1: Заполнение KPI-счётчика
метод ЗаполнитьКарточку<Имя>()
    знч Данные = Получить<Имя>()
    Компоненты.НадписьЗначение1.Значение = Данные.<Поле1>.Представление()
    Компоненты.НадписьЗначение2.Значение = "${Данные.<Поле2>} руб."
;

// P2: Навигация по клику на карточку
метод Карточка<Раздел>ПриНажатии(Источник: Карточка, Событие: СобытиеПриНажатии)
    <Раздел>ФормаСписка.Открыть()
;

// P2: Открыть форму объекта в диалоге (кнопка «Создать»)
метод Создать<Объект>ПриНажатии(Команда: ОбычнаяКоманда)
    <Объект>ФормаОбъекта.Открыть(СпособОткрытия = СпособОткрытияФормы.ВДиалоговомОкне)
;

// P3: Заполнение таблицы
метод ЗаполнитьТаблицу<Имя>()
    Компоненты.Таблица<Имя>.Источник.Данные = Получить<Имя>()
;

// P5: Обновление диаграммы
метод ОбновитьДиаграммы()
    Компоненты.<ИмяДиаграммы>.ОбновитьДиаграмму()
;

// Серверный метод (данные с сервера):
@НаСервере @ДоступноСКлиента
статический метод Получить<Имя>(): <ТипВозврата>
    возврат <Объект>.Получить<Имя>()
;
```

## Критические правила .xbsl

- **НЕ** использовать `@КомпонентИнтерфейса` — методы на верхнем уровне файла
- `@Обработчик` — только для `ПослеСоздания`
- Обращение к компонентам: `Компоненты.<ИмяКомпонента>.Значение`
- Серверные данные: `@НаСервере @ДоступноСКлиента статический метод`
- Импорт подсистем: `импорт <ИмяПодсистемы>` (строчные буквы в слове `импорт`, без пробела перед именем)
- Цикл по результатам запроса: `для Строка из Запрос.Выполнить() ... ;` (без `знч`, без `цикл/конец`)
- Инициализация структуры: `новый Тип(Поле1 = Значение1, Поле2 = Значение2)`
- Nullable поля в запросе: `.ЗаменитьNull()` для Строка?, `.ЗаменитьNull(0)` для Число?
