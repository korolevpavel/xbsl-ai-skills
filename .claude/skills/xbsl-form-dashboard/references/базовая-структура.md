# Базовая структура ГлавнаяСтраница.yaml

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: ГлавнаяСтраница
ОбластьВидимости: ВПодсистеме   # или ВПроекте — если ссылается на форму из другой подсистемы
Импорт:
    - <ИмяПодсистемы>          # подсистемы, из которых берутся данные (только если нужны)
Наследует:
    Тип: Форма
    ВключатьВАвтоИнтерфейс: Ложь
    ПовторноеИспользованиеФормы: Истина
    Содержимое:
        Тип: ПроизвольныйШаблонФормы
        Содержимое:
            Тип: Группа
            РастягиватьПоВертикали: Ложь
            ШиринаВКолонках: Четверная
            Содержимое:
                # виджеты (карточки) — см. паттерны p1–p9
```

**Обязательные правила:**
- `ВключатьВАвтоИнтерфейс: Ложь` — **всегда**, без исключений
- `ПовторноеИспользованиеФормы: Истина` — **всегда** (кэширование)
- `ШиринаВКолонках: Четверная` на корневой Группе — 4-колонная сетка
- `Импорт:` — только если берёте данные из другой подсистемы

---

## Правила компоновки карточек

**ШиринаВКолонках** (внутри корневой Четверной группы):
- `Одинарная` — 1/4 ширины (компактный KPI)
- `Двойная` — 1/2 ширины (стандарт для большинства карточек)
- `Четверная` — полная ширина (баннеры, широкие таблицы)

**МинимальнаяВысота: 230** — стандарт для KPI и карточек с таблицами.

**РастягиватьПоГоризонтали: Ложь** на карточке — используется `ШиринаВКолонках` для ширины.
