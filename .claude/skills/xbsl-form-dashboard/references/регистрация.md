# Регистрация в навигационной панели

Форма **не появляется автоматически** — нужно зарегистрировать в навигации.

> Для `Тип: Форма` нет автокоманды `.Открыть` (в отличие от Справочников/Документов).
> Нужно использовать `НавигационнаяКоманда` с `ТипФормы:`, не `=ГлавнаяСтраница.Открыть`.

---

## Шаг 1: Проверить наличие `Приложение*.yaml`

Если в проекте нет `КомпонентИнтерфейса` с `Тип: СтандартноеКлиентскоеПриложениеСРазделами`
— платформа строит авто-навигацию и кастомную панель не подключает. Нужно создать его.

```yaml
ВидЭлемента: КомпонентИнтерфейса
Ид: <UUID>
Имя: Приложение
ОбластьВидимости: ВПодсистеме
Импорт:
    - <ДругаяПодсистема>    # если нужны =Объект.ОткрытьСписок из неё
Наследует:
    Тип: СтандартноеКлиентскоеПриложениеСРазделами
    Заголовок: <Заголовок приложения>
    ОриентацияПанелиНавигации: Вертикальная
    ОсобоеОформлениеПервогоРаздела: Истина
    КомандныйИнтерфейсПанелиНавигации:
        Тип: ФрагментКомандногоИнтерфейса
        Элементы:
            -
                Тип: НавигационнаяКоманда
                Представление: Главная
                Изображение: Плюс.svg
                ТипФормы: <ИмяФормы>
            -
                Тип: ГруппаКомандногоИнтерфейса
                Представление: <Раздел>
                Элементы:
                    - =<Объект>.ОткрытьСписок
```

## Шаг 2: Если уже есть `ПанельНавигации.yaml`

Добавить в `ПанельНавигации.yaml`:
```yaml
-
    Тип: НавигационнаяКоманда
    Изображение: Плюс.svg
    Представление: Главная
    ТипФормы: <ИмяФормы>
```

Убедиться что `Приложение*.yaml` ссылается на неё через:
```yaml
КомандныйИнтерфейсПанелиНавигации: =ПанельНавигации
```

## Когда нужна отдельная подсистема Дашборд

Если форма обращается к данным из подсистем с **круговыми зависимостями**
(например, `Склад` использует `Основное` → `Основное` не может импортировать `Склад`),
нужно создать отдельную подсистему `Дашборд` на верхнем уровне:

```yaml
# Дашборд/Подсистема.yaml
Использование:
    - Основное
    - Склад
    - Закупки
Интерфейс:
    ВключатьВАвтоИнтерфейс: Ложь
```

Затем перенести в `Дашборд/`:
- `ГлавнаяСтраница.yaml` + `ГлавнаяСтраница.xbsl`
- `Приложение.yaml` (добавить `Импорт: - Основное`)
- `ПанельНавигации.yaml`
- Вспомогательные структуры (`СтрокаОстатков.yaml` и т.п.)

**Структуры в подсистеме Дашборд:**
```yaml
ВидЭлемента: Структура
Имя: СтрокаОстатков
ОбластьВидимости: ВПроекте    # ← обязательно если используется в другой подсистеме
Окружение: КлиентИСервер      # ← обязательно для Структур
Поля:
    - Имя: Наименование
      Тип: Строка
    - Имя: Количество
      Тип: Число
```

**Правило:** `ОбластьВидимости: ВПроекте` у формы, если `Приложение.yaml` находится
в другой подсистеме и ссылается на `ТипФормы: ГлавнаяСтраница`.
