# Стилизация: цвета, шрифты, условная видимость

## Цвета иконок (палитра стилевых цветов)

```yaml
Цвет: =Цвета.Оранжевый
Цвет: =Цвета.Зеленый
Цвет: =Цвета.Голубой
Цвет: =Цвета.Красный
Цвет: =Цвета.Фиолетовый
```

## Фон карточек

```yaml
# Стилевые (предпочтительно — адаптируются к теме):
Фон: =Цвета.Стилевые.ФонПервичный
Фон: =Цвета.Стилевые.ФонВторичный

# Кастомный цвет (для лендингов и баннеров):
Фон:
    Тип: АбсолютныйЦвет
    Значение: RGB(5955FF)
```

## Шрифты

```yaml
# Стилевые (предпочтительно — адаптируются к теме):
Шрифт: =СтилевыеШрифты.ЗаголовокОбычный
Шрифт: =СтилевыеШрифты.ЗаголовокКрупный
Шрифт: =СтилевыеШрифты.ОченьКрупный
Шрифт: =СтилевыеШрифты.Крупный

# Абсолютный (для кастомного дизайна баннеров):
ШрифтЗаголовка:
    Тип: АбсолютныйШрифт
    Полужирный: Истина
    Размер: 36

ШрифтСодержимого: =СтилевыеШрифты.ОченьКрупный
```

## Цвет текста (для цветных карточек и баннеров)

```yaml
ЦветЗаголовка:
    Тип: АбсолютныйЦвет
    Значение: RGB(FFFFFF)
ЦветСодержимого:
    Тип: АбсолютныйЦвет
    Значение: RGB(FFFFFF)
```

## Условная видимость команд по платформе

```yaml
ДополнительныеКоманды:
    Тип: ФрагментКомандногоИнтерфейса
    Элементы:
        -
            Тип: ОбычнаяКоманда
            Видимость: =КлиентскоеУстройство.ВидПлатформы == ВидПлатформыКлиента.Веб
            Изображение: ОткрытьСсылку.svg
            Обработчик: <Обработчик>ПриНажатии
            Представление: <Текст>
```

**Правило:** предпочитать стилевые цвета и шрифты (`=Цвета.Стилевые.*`, `=СтилевыеШрифты.*`) —
они адаптируются к теме приложения. Абсолютные (`АбсолютныйЦвет`, `АбсолютныйШрифт`) — только
для баннеров и лендингов с фиксированным дизайном.
