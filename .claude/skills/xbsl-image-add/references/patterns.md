# Паттерны xbsl-image-add

## P5 — Простое фото (без серверной обработки)

### Реквизит объекта

```yaml
-
    Ид: <UUID>
    Имя: Фото
    Тип: ДвоичныйОбъект.Ссылка?
```

### Компоненты формы

```yaml
-
    Тип: Группа
    Содержимое:
        -
            Тип: Картинка
            Изображение: =Объект.Фото ?? Ресурс{Аккаунт.svg}.Ссылка
            МаксимальнаяВысота: 132
            Масштабирование: Пропорционально
            РастягиватьПоВертикали: Истина
        -
            Тип: ВыборФайлов
            Имя: ФотоВыборФайлов
            МаксимальнаяВысота: 32
            МножественныйВыбор: Ложь
            Описание: Выбор фото
            ПриИзменении: ПриВыбореФото
            РастягиватьПоВертикали: Истина
            Фильтр: image/*
```

### Обработчик (.xbsl)

```xbsl
метод ПриВыбореФото(Источник: ВыборФайлов, Событие: СобытиеПриИзменении<ЧитаемыйМассив<ВыбранныйФайл>>)
    знч ВыбранныйФайл = Событие.НовоеЗначение.Первый()
    знч Изображение = ЗагрузкаФайлов.НачатьЗагрузку(ВыбранныйФайл).ПолучитьРезультат()
    Объект.Фото = Изображение
;
```

## P6 — Фото с обработкой (ресайз / обрезка по кругу)

### Реквизит объекта

Тот же, что в P5.

### Компоненты формы

Те же, что в P5.

### Обработчик (.xbsl)

```xbsl
метод ПриВыбореФото(Источник: ВыборФайлов, Событие: СобытиеПриИзменении<ЧитаемыйМассив<ВыбранныйФайл>>)
    знч ВыбранныйФайл = Событие.НовоеЗначение.Первый()
    знч Изображение = ЗагрузкаФайлов.НачатьЗагрузку(ВыбранныйФайл).ПолучитьРезультат()
    Объект.Фото = ОбработатьФото(Изображение)
;

@НаСервере @ДоступноСКлиента
статический метод ОбработатьФото(Фото: ДвоичныйОбъект.Ссылка): ДвоичныйОбъект.Ссылка
    знч ИсходноеФото = Фото.Загрузить()
    исп ИсходноеПоток = ИсходноеФото.ОткрытьПотокЧтения()
    пер Изображение = новый ОбрабатываемоеИзображение(ИсходноеПоток)
    Изображение = Изображение
        .УстановитьФормат(ФорматИзображения.Png)
        .УстановитьРазмер(200, 200)
        .ОбрезатьПоКругу()    // убрать если круглая обрезка не нужна
    знч Данные = Изображение.ПолучитьИзображение()
    возврат ОбъектноеХранилище.Загрузить(ИсходноеФото.ИмяФайла, Данные).Ссылка
;
```

## Аватар в форме списка

Добавить **перед** первой стандартной колонкой в `Колонки`:

```yaml
-
    Тип: ПроизвольнаяКолонкаТаблицы<СтрокаДинамическогоСписка<...<ИмяФормы>.ДанныеСтрокиСписка>>
    МаксимальнаяШирина: 68
    Заголовок: Фото
    Содержимое:
        Тип: Картинка
        Высота: 40
        Изображение: =ДанныеСтроки.Данные.Фото ?? Ресурс{Аккаунт.svg}.Ссылка
        Фигура: Круглая
```

Также добавить поле в `Свойства.Список.ЗначениеПоУмолчанию.Поля`:

```yaml
-
    Тип: ПолеДинамическогоСписка
    Выражение: Фото
```

## Справка по параметрам ВыборФайлов

| Параметр | Тип | Описание |
|----------|-----|---------|
| `Фильтр` | `Строка` | MIME-тип: `image/*`, `image/jpeg` и т.д. (строка, не массив!) |
| `МножественныйВыбор` | `Булево` | Всегда `Ложь` для фото |
| `МаксимальнаяВысота` | `Число` | Высота кнопки загрузки (обычно 32) |
| `ПриИзменении` | `Строка` | Имя метода-обработчика в `.xbsl` |
| `Описание` | `Строка` | Подпись кнопки |

## API ОбрабатываемоеИзображение

Все методы возвращают `ОбрабатываемоеИзображение` (fluent API), кроме `ПолучитьИзображение`.

| Метод | Описание |
|-------|---------|
| `УстановитьФормат(ФорматИзображения)` | `Png`, `Jpeg`, `Gif`, `Bmp`, `Tiff` |
| `УстановитьРазмер(Ш, В)` | Точный размер в пикселях |
| `УстановитьШирину(Ш)` | Только ширина, высота пропорционально |
| `УстановитьВысоту(В)` | Только высота, ширина пропорционально |
| `Масштабировать(Множитель)` | Нельзя совмещать с `УстановитьРазмер` |
| `ОбрезатьПоКругу()` | По центру, диаметр = min(Ш, В) |
| `Повернуть(Угол)` | По часовой стрелке, в градусах |
| `ВыбратьОбласть(X, Y, Ш, В)` | Выбрать регион исходного изображения |
| `ВОттенкиСерого()` | Grayscale |
| `ПолучитьИзображение()` | **Финальный вызов** — возвращает `Байты` |
