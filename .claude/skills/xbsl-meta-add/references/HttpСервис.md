# Спецификация: HttpСервис

## Дефолты

- **Имя** — PascalCase, описывает ресурс + `HttpСервис` или `АпиСервис` (`КонтрагентыHttpСервис`, `СделкиАпи`)
- **Видимость** — `ВПодсистеме`
- **КорневойUrl** — `/api/<ресурс>` (например `/api/contractors`)

## UUID

Нужен: 1 (объект). URL-шаблоны и методы UUID не требуют.

## Структура YAML

### Минимальная:

```yaml
ВидЭлемента: HttpСервис
Ид: <UUID>
Имя: <ИмяСервиса>
ОбластьВидимости: ВПодсистеме
КорневойUrl: /api/<ресурс>
ШаблоныUrl:
    - Имя: Список
      Шаблон: /
      Методы:
          - Метод: GET
            Обработчик: ПолучитьСписок
          - Метод: POST
            Обработчик: Создать
    - Имя: ЭлементПоИд
      Шаблон: /{id}
      Методы:
          - Метод: GET
            Обработчик: ПолучитьПоИд
          - Метод: PUT
            Обработчик: Обновить
          - Метод: DELETE
            Обработчик: Удалить
```

### Полная (с контролем доступа):

```yaml
ВидЭлемента: HttpСервис
Ид: <UUID>
Имя: <ИмяСервиса>
ОбластьВидимости: ВПодсистеме
КорневойUrl: /api/<ресурс>
КонтрольДоступа:                  # необязательно — доступ на уровне сервиса
    Разрешения:
        Вызов: РазрешеноАутентифицированным  # или РазрешеноВсем, РазрешеноАдминистраторам
ШаблоныUrl:
    - Имя: <ИмяШаблона>
      Шаблон: /{id}/вложенный      # параметры пути в фигурных скобках
      КонтрольДоступа:             # необязательно — переопределяет уровень сервиса
          Разрешения:
              Вызов: РазрешеноАдминистраторам
      ЛюбойМетод: <ИмяОбработчика> # необязательно — один обработчик для всех методов
      Методы:
          - Метод: GET | POST | PUT | DELETE | PATCH | HEAD | OPTIONS
            Обработчик: <ИмяМетодаВXBSL>
            КонтрольДоступа:       # необязательно — переопределяет уровень шаблона
                Разрешения:
                    Вызов: РазрешеноВсем
```

## Правила

- `КорневойUrl` — базовый путь для всех шаблонов
- `Шаблон` — относительный путь; параметры в `{фигурных_скобках}`, доступны через `Запрос.Параметры`
- `ЛюбойМетод` — обработчик вызывается для любого HTTP-метода (альтернатива `Методы`)
- `КонтрольДоступа` — иерархический: метод → шаблон → сервис

## XBSL паттерны

```xbsl
// Получение списка с параметром из query string
метод ПолучитьСписок(Запрос: HttpСервисЗапрос)
    попытка
        пер Ограничение: Число = 100
        знч ПараметрЛимит = Запрос.Параметры.ПолучитьПервый("limit")
        если ПараметрЛимит != Неопределено
            Ограничение = Мин(новый Число(ПараметрЛимит), 100)
        ;
        знч Данные = <Справочник>.ПолучитьСписок(Ограничение)
        Запрос.Ответ.Заголовки.Установить("Content-Type", "application/json")
        Запрос.Ответ.УстановитьТело(СериализацияJson.ЗаписатьОбъект(Данные))
    поймать Исключение: Исключение
        _ОбработатьОшибку(Запрос.Ответ, Исключение)
    ;
;

// Создание объекта из тела запроса
метод Создать(Запрос: HttpСервисЗапрос)
    попытка
        знч Данные = СериализацияJson.ПрочитатьОбъект(Запрос.Тело, Тип<Данные<Справочника>>)
        знч Ссылка = <Справочник>.Создать(Данные)
        Запрос.Ответ.УстановитьКодСтатуса(201)
        Запрос.Ответ.УстановитьТело(Ссылка.Ид.ВСтроку())
    поймать Исключение: Исключение
        _ОбработатьОшибку(Запрос.Ответ, Исключение)
    ;
;

// Получение по id (параметр пути — через Запрос.Параметры, не ПараметрыПути)
метод ПолучитьПоИд(Запрос: HttpСервисЗапрос)
    попытка
        знч Ид = Запрос.Параметры.ПолучитьПервый("id")
        знч Объект = <Справочник>.НайтиПоИд(Ид)
        если Объект == Неопределено
            Запрос.Ответ.УстановитьКодСтатуса(404)
            возврат
        ;
        Запрос.Ответ.УстановитьТело(СериализацияJson.ЗаписатьОбъект(Объект))
    поймать Исключение: Исключение
        _ОбработатьОшибку(Запрос.Ответ, Исключение)
    ;
;

// Вспомогательный метод обработки ошибок
метод _ОбработатьОшибку(Ответ: HttpСервисОтвет, Исключение: Исключение)
    Ответ.УстановитьКодСтатуса(500)
    Ответ.УстановитьТело(Исключение.Описание)
;
```
