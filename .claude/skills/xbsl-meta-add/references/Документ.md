# Спецификация: Документ

## Дефолты

- **Имя** — PascalCase, существительное во **множественном** числе (`Заказы`, `РасходныеНакладные`, `ЗаявкиНаРемонт`)
- **Видимость** — всегда `ВПодсистеме`. Менять на `ВПроекте` только если пользователь явно попросил
- **РежимУдаления** — `Немедленно`
- **Автонумерация** — добавляй при явном запросе или если очевидно нужна (заказы, накладные)
- **ТабличныеЧасти** — только если пользователь упомянул

## UUID

Нужны: 1 (объект) + по 1 на каждый пользовательский реквизит + по 1 на каждую табличную часть и её реквизиты.
Системные поля `Дата`, `Номер`, `Файлы` — **без UUID**.

## Табличные части

Если объект содержит ТЧ — прочитай `references/ТабличныеЧасти.md`.

## Правила

- `Дата`, `Номер`, `Файлы` — системные реквизиты, **без `Ид`**
- `Файлы` — **без `Тип`**
- `Номер` — `Тип: Строка` можно опустить, если указана только `Автонумерация`
- Блок `Интерфейс` можно опустить, если документ управляется программно
- `НезаполненноеЗначение: ЗапретитьВсегда` — не ставится на системные реквизиты

## XBSL паттерны

### Бизнес-метод (сумма по табличной части):

```xbsl
@ВПроекте
метод ПолучитьСумму(): Число
    знч Запрос = Запрос{
        ВЫБРАТЬ
            СУММА(Товары.Сумма) КАК Итого
        ИЗ
            <ИмяДокумента>.Товары КАК Товары
        ГДЕ
            Товары.Владелец == %этот
    }
    возврат Запрос.Выполнить().Первый()?.Итого ?? 0
;
```

### Обработчик заполнения (вызывается при создании нового объекта):

```xbsl
@Обработчик
метод ПриЗаполнении(ТекстЗаполнения: Строка)
    Ответственный = ПользователиКлиентИСервер.ПолучитьТекущегоСотрудника()
    Статус = <ИмяПеречисления>.<ЗначениеПоУмолчанию>
;
```

### Обработчик создания на основании:

```xbsl
@Обработчик
метод ПриСозданииНаОсновании(Основание: <ИмяДокумента>.Ссылка)
    Владелец = Основание
    Контрагент = Основание.Контрагент
;
```

### Контроль доступа:

```xbsl
@Обработчик
метод ВычислитьРазрешенияДоступа(): ЧитаемаяКоллекция<РазрешениеДоступа>
    возврат [
        новый РазрешениеДоступа(
            [новый КлючДоступаДляАутентифицированных.Объект()],
            [Сущность.Право.Чтение, Сущность.Право.Создание, Сущность.Право.Изменение, Сущность.Право.Удаление]
        )
    ]
;
```
