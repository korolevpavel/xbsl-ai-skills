# Спецификация: Перечисление

## Дефолты

- **Имя** — PascalCase, существительное в **единственном** числе (`СтатусЗадачи`, `ПриоритетЗаявки`, `ВидСвязи`)
- **Видимость** — всегда `ВПодсистеме`. Менять на `ВПроекте` только если пользователь явно попросил
- **Методы** — не создавай без явного запроса

## UUID

Нужны: 1 (объект) + по 1 на каждый элемент.

## Структура YAML

### Минимальная:

```yaml
ВидЭлемента: Перечисление
Ид: <UUID>
Имя: <ИмяПеречисления>
ОбластьВидимости: ВПодсистеме
Элементы:
    -
        Ид: <UUID>
        Имя: <Элемент1>
    -
        Ид: <UUID>
        Имя: <Элемент2>
```

### Полная (с представлениями и значением по умолчанию):

```yaml
ВидЭлемента: Перечисление
Ид: <UUID>
Имя: <ИмяПеречисления>
ОбластьВидимости: ВПодсистеме
Элементы:
    -
        Ид: <UUID>
        Имя: <Элемент1>
        Представление: $ЛокализованныеСтроки.<Элемент1>  # или просто строка: "Новая"
        ПоУмолчанию: Истина
    -
        Ид: <UUID>
        Имя: <Элемент2>
        Представление: $ЛокализованныеСтроки.<Элемент2>
```

## Правила

- `ОбластьВидимости: ВПроекте` — если перечисление используется в нескольких подсистемах
- `Представление` у элемента — два допустимых формата:
  - Локализованная ссылка: `$ЛокализованныеСтроки.КлючСтроки`
  - Простая строка: `"OK"`, `"Bad Request"` (для технических/кодовых значений)
- `ПоУмолчанию: Истина` — максимум у одного элемента

## XBSL паттерны

Аннотация `@ВПроекте` / `@ВПодсистеме` должна совпадать с `ОбластьВидимости` в YAML.

### Метод с цветом (самый частый):

```xbsl
@<ОбластьВидимости>
метод ПолучитьЦвет(): Цвет | Авто
    выбор этот
    когда <Элемент1>
        возврат Цвета.Желтый
    когда <Элемент2>
        возврат Цвета.Зеленый
    иначе
        возврат Авто
    ;
;
```

### Метод с подсказкой:

```xbsl
@<ОбластьВидимости>
метод ПолучитьПодсказку(): Строка?
    выбор этот
    когда <Элемент1>
        возврат "<текст>"
    иначе
        возврат Неопределено
    ;
;
```

Доступные цвета: `Цвета.Красный`, `Цвета.Зеленый`, `Цвета.Синий`, `Цвета.Желтый`, `Цвета.Голубой`, `Цвета.Фиолетовый`.

`выбор этот` — перебирает элементы перечисления; имена используются без кавычек.
