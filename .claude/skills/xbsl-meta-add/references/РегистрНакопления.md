# Спецификация: РегистрНакопления

## Дефолты

- **Имя** — PascalCase, существительное во множественном числе (`Продажи`, `ОстаткиНоменклатуры`)
- **Видимость** — `ВПодсистеме`
- **ВидРегистра** — по умолчанию `Остатки`. Уточни у пользователя если не указан явно.
- **Регистратор** — **обязателен**: без него проект не соберётся. Если пользователь не указал регистратора явно — **остановись и спроси**:
  > Какой документ (или документы) будет регистратором для этого регистра?
  > Укажи один или несколько документов. Без регистратора сборка не пройдёт.
  Только после ответа продолжай создание файла.

## UUID

Нужны: 1 (объект) + по 1 на каждое Измерение + по 1 на каждый Ресурс + по 1 на каждый Реквизит (кроме Регистратора — он системный, без Ид).

        Тип: <Справочник>.Ссылка?

        ДлинаЦелойЧасти: 15
        ДлинаДробнойЧасти: 2

## Правила

### Измерения
- Список измерений **не может быть пустым**.
- **Запрещённые типы для измерений**: `Строка` и `Байты` неограниченной длины, `ДвоичныеДанныеСсылка`.
- Измерения имеют те же свойства, что и реквизиты.

### Ресурсы
- Список ресурсов **не может быть пустым**.
- Ресурсы — только числовые показатели. Имеют те же свойства, что и реквизиты.

### ВидРегистра
- `Остатки` (по умолчанию) — хранит **и итоговые значения (остатки), и обороты**. Виртуальные таблицы: `.Остатки`, `.Обороты`, `.ОстаткиИОбороты`. При добавлении записей **обязателен** `ВидЗаписи = ВидЗаписиРегистраНакопления.Приход` или `.Расход`.
- `Обороты` — хранит только обороты. Виртуальная таблица: `.Обороты`. Понятие `ВидЗаписи` (Приход/Расход) **не используется**.

### Регистратор
- Системный реквизит, **без `Ид`**.
- Тип — один документ (`<Документ>.Ссылка?`) или union нескольких (`<ДокА>.Ссылка|<ДокБ>.Ссылка|?`).
- **Без Регистратора проект не соберётся.**

### ОбластьВидимости
- `ВПодсистеме` — виден внутри одной подсистемы (по умолчанию).
- `ВПроекте` — виден во всех подсистемах проекта.
- `Глобально` — виден во всех подсистемах всех проектов.

## XBSL паттерны

### Запись движений (Приход) — регистр Остатки:

```xbsl
метод ВыполнитьПроведение()
    знч НаборЗаписей = новый <ИмяРегистра>.НаборЗаписей()
    НаборЗаписей.Фильтр.Установить(Регистратор = Ссылка)
    для Строка из <ТабличнаяЧасть>
        НаборЗаписей.ДобавитьЗапись(
            Период = Дата,
            ВидЗаписи = ВидЗаписиРегистраНакопления.Приход,
            <Измерение> = Строка.<Измерение>,
            <Ресурс> = Строка.<Ресурс>
        )
    ;
    НаборЗаписей.Записать()
;

метод ОтменитьПроведение()
    знч НаборЗаписей = новый <ИмяРегистра>.НаборЗаписей()
    НаборЗаписей.Фильтр.Установить(Регистратор = Ссылка)
    НаборЗаписей.Записать()   // пустой набор — удаляет все движения по регистратору
;
```

### Запись движений (Расход) — регистр Остатки:

```xbsl
НаборЗаписей.ДобавитьЗапись(
    Период = Дата,
    ВидЗаписи = ВидЗаписиРегистраНакопления.Расход,
    <Измерение> = ...,
    <Ресурс> = ...
)
```

### Запись движений — регистр Обороты (ВидЗаписи не указывается):

```xbsl
НаборЗаписей.ДобавитьЗапись(
    Период = Дата,
    <Измерение> = Строка.<Измерение>,
    <Ресурс> = Строка.<Ресурс>
)
```

### Запрос остатков (ВидРегистра: Остатки):

```xbsl
@ВПодсистеме
метод ПолучитьОстаток(<Измерение>: <Справочник>.Ссылка): Число
    знч Запрос = Запрос{
        ВЫБРАТЬ
            Таблица.<Ресурс>Остаток КАК <Ресурс>
        ИЗ
            <ИмяРегистра>.Остатки КАК Таблица
        ГДЕ
            Таблица.<Измерение> == %<Измерение>
    }
    возврат Запрос.Выполнить().ПервыйИлиНеопределено()?.<Ресурс> ?? 0
;
```

### Запрос оборотов (ВидРегистра: Обороты):

```xbsl
знч Запрос = Запрос{
    ВЫБРАТЬ
        Таблица.<Измерение>,
        СУММА(Таблица.<Ресурс>) КАК Итого
    ИЗ
        <ИмяРегистра>.Обороты(%НачалоПериода, %КонецПериода) КАК Таблица
    СГРУППИРОВАТЬ ПО
        Таблица.<Измерение>
}
```
