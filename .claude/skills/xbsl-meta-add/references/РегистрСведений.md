# Спецификация: РегистрСведений

## Дефолты

- **Имя** — PascalCase, существительное во множественном числе (`КурсыВалют`, `ЦеныНоменклатуры`)
- **Видимость** — `ВПодсистеме`
- **Периодичность** — `Непериодический` (если не нужна история по времени)

## UUID

Нужны: 1 (объект) + по 1 на каждое Измерение + по 1 на каждый Ресурс + по 1 на каждый Реквизит.

## Структура YAML

### Минимальная:

```yaml
ВидЭлемента: РегистрСведений
Ид: <UUID>
Имя: <ИмяРегистра>
ОбластьВидимости: ВПодсистеме
Периодичность: Непериодический
Измерения:
    - Ид: <UUID>
      Имя: <ИмяИзмерения>
      Тип: <Справочник>.Ссылка?
      Ведущее: Истина
Ресурсы:
    - Ид: <UUID>
      Имя: <ИмяРесурса>
      Тип: Число
      ДлинаЦелойЧасти: 15
      ДлинаДробнойЧасти: 4
```

### Полная:

```yaml
ВидЭлемента: РегистрСведений
Ид: <UUID>
Имя: <ИмяРегистра>
ОбластьВидимости: ВПодсистеме
Периодичность: День           # Непериодический | День | Месяц | Момент
Интерфейс:
    ВключатьВАвтоИнтерфейс: Ложь
    Список:
        Представление: <МножественноеЧисло>
        Форма: <ИмяФормыСписка>
КонтрольДоступа:
    Разрешения:
        ПоУмолчанию: РазрешеноВсем   # или РазрешенияВычисляютсяДляКаждогоОбъекта
    РасчетРазрешенийПо:
        - <ИмяИзмерения>
Измерения:
    - Ид: <UUID>
      Имя: <ИмяИзмерения>
      Тип: <Справочник>.Ссылка?
      Ведущее: Истина              # ведущие измерения — часть ключа записи
      НезаполненноеЗначение: ЗапретитьВсегда
Ресурсы:
    - Ид: <UUID>
      Имя: <ИмяРесурса>
      Тип: Число
      ДлинаЦелойЧасти: 15
      ДлинаДробнойЧасти: 4
      МинимальноеЗначение: 0
Реквизиты:                     # необязательно — дополнительные поля записи
    - Ид: <UUID>
      Имя: <ИмяРеквизита>
      Тип: <Тип>
```

## Правила

- `Периодичность` — определяет, хранится ли история: `Непериодический` (одна запись на ключ), `День` / `Месяц` / `Момент` (история по времени)
- `Ведущее: Истина` — минимум одно измерение должно быть ведущим; ведущие измерения образуют первичный ключ
- `Измерения` — определяют ключ (по чему хранятся данные)
- `Ресурсы` — хранимые значения (что хранится)
- `Реквизиты` — дополнительные поля записи, не входящие в ключ

## XBSL паттерны

### Запрос среза последних (для периодических регистров):

```xbsl
@ВПодсистеме
метод ПолучитьПоследнееЗначение(Измерение: <Справочник>.Ссылка): Число?
    знч Запрос = Запрос{
        ВЫБРАТЬ ПЕРВЫЕ 1
            Таблица.<Ресурс> КАК Значение
        ИЗ
            <ИмяРегистра>.СрезПоследних() КАК Таблица
        ГДЕ
            Таблица.<Измерение> == %Измерение
    }
    возврат Запрос.Выполнить().ПервыйИлиНеопределено()?.Значение
;
```

### Удаление записи по ключу:

```xbsl
знч КлючЗаписи = новый <ИмяРегистра>.КлючЗаписи(
    <Измерение> = Значение
)
Удалить(КлючЗаписи)
```
