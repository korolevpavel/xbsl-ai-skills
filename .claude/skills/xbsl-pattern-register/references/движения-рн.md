# Паттерны: движения по регистру накопления

## Правило: именование файла обработчика

Код обработчиков документа пишется в файл **`<ИмяДокумента>.Объект.xbsl`** (не `<ИмяДокумента>.xbsl`).

Без суффикса `.Объект` платформа не привязывает код к объекту — переменные `Ссылка`, `Дата`, реквизиты будут недоступны, а метод `ПослеЗаписи` не найден. Компилятор выдаст:
```
Variable "Ссылка" is not defined
A handler associated with method "ПослеЗаписи" is not found
```

---

## Именование в виртуальной таблице `.Остатки`

| Тип поля | В YAML (имя) | В запросе к `.Остатки` |
|---|---|---|
| Измерение | `<Имя>` | `Остатки.<Имя>` (без суффикса) |
| Ресурс | `<Имя>` | `Остатки.<ИмяОстаток>` (суффикс `Остаток`) |

---

## Паттерн A1: Приход по РН Остатки

```xbsl
@Обработчик
метод ПослеЗаписи(До: <Документ>.Данные, ПараметрыЗаписи: <Документ>.ПараметрыЗаписи)
    знч НаборЗаписей = новый <ИмяРегистра>.НаборЗаписей()
    НаборЗаписей.Фильтр.Установить(Регистратор = Ссылка)

    для Строка из <ТабличнаяЧасть>
        НаборЗаписей.ДобавитьЗапись(
            Период = Дата,
            ВидЗаписи = ВидЗаписиРегистраНакопления.Приход,
            <Измерение1> = <ЗначениеИзмерения1>,   // например: Склад = Склад
            <Измерение2> = Строка.<Измерение2>,     // например: Номенклатура = Строка.Номенклатура
            <Ресурс> = Строка.<Ресурс>              // например: Количество = Строка.Количество
        )
    ;

    НаборЗаписей.Записать()
;
```

---

## Паттерн A2: Расход по РН Остатки

Отличается только `ВидЗаписи`:

```xbsl
НаборЗаписей.ДобавитьЗапись(
    Период = Дата,
    ВидЗаписи = ВидЗаписиРегистраНакопления.Расход,
    <Измерение1> = <ЗначениеИзмерения1>,
    <Измерение2> = Строка.<Измерение2>,
    <Ресурс> = Строка.<Ресурс>
)
```

---

## Паттерн A3: Движение по РН Обороты

`ВидЗаписи` **не указывается** — регистр оборотов не имеет понятия приход/расход:

```xbsl
НаборЗаписей.ДобавитьЗапись(
    Период = Дата,
    <Измерение1> = <ЗначениеИзмерения1>,
    <Измерение2> = Строка.<Измерение2>,
    <Ресурс1> = Строка.<Ресурс1>,
    <Ресурс2> = Строка.<Ресурс2>
)
```

---

## Паттерн A4: Очистка движений (автоматически через фильтр)

**`ОтменитьПроведение` в XBSL не существует** — такого обработчика нет. Компилятор выдаст:
```
A handler associated with method "ОтменитьПроведение" is not found
```

Очистка старых движений происходит **автоматически** в `ПослеЗаписи` благодаря установке фильтра до `Записать()`. При повторном проведении документа старые записи удаляются и заменяются новыми. При удалении документа платформа очищает движения сама.

Никакого отдельного обработчика для отмены проведения писать не нужно — паттерны A1–A3 уже содержат правильную логику очистки через `НаборЗаписей.Фильтр.Установить(Регистратор = Ссылка)` перед `Записать()`.

---

## Паттерн A5: Движение сразу по двум регистрам

Если один документ пишет в несколько регистров — создай отдельный НаборЗаписей для каждого:

```xbsl
@Обработчик
метод ПослеЗаписи(До: <Документ>.Данные, ПараметрыЗаписи: <Документ>.ПараметрыЗаписи)
    знч НаборОстатки = новый <РегистрОстатки>.НаборЗаписей()
    НаборОстатки.Фильтр.Установить(Регистратор = Ссылка)

    знч НаборОбороты = новый <РегистрОбороты>.НаборЗаписей()
    НаборОбороты.Фильтр.Установить(Регистратор = Ссылка)

    для Строка из <ТабличнаяЧасть>
        НаборОстатки.ДобавитьЗапись(
            Период = Дата,
            ВидЗаписи = ВидЗаписиРегистраНакопления.Расход,
            <Измерения и ресурсы>
        )
        НаборОбороты.ДобавитьЗапись(
            Период = Дата,
            <Измерения и ресурсы>
        )
    ;

    НаборОстатки.Записать()
    НаборОбороты.Записать()
;
```

## Паттерн B: Контроль остатков после записи

Добавляется **после** `НаборЗаписей.Записать()`. Платформа выполнит откат транзакции при выброске исключения.

```xbsl
    знч Запрос = Запрос{
        ВЫБРАТЬ
            Остатки.<Измерение>,              // без суффикса: Остатки.Номенклатура
            Остатки.<РесурсОстаток>           // с суффиксом: Остатки.КоличествоОстаток
        ИЗ
            <ИмяРегистра>.Остатки КАК Остатки
        ГДЕ
            Остатки.<ИзмерениеФильтр> == %<РеквизитДокумента>   // например: Остатки.Склад == %Склад
            И Остатки.<РесурсОстаток> < 0
    }

    исп Результат = Запрос.Выполнить()

    пер СтрокаСообщения = ""

    для Выборка из Результат
        СтрокаСообщения += "Не хватает ${Выборка.<Измерение>} в количестве %{-Выборка.<РесурсОстаток>}\н"
    ;

    если не СтрокаСообщения.Пусто()
        выбросить новый ИсключениеНедопустимоеСостояние(Описание = СтрокаСообщения)
    ;
```

### Правила паттерна B

- Фильтр в `ГДЕ` по измерениям нужен только если хочешь ограничить проверку (например, только по текущему складу). Без фильтра — проверяются все остатки в регистре.
- Измерения в виртуальной таблице — **без суффикса**: `Остатки.Склад`, `Остатки.Номенклатура`
- Ресурсы в виртуальной таблице — **с суффиксом `Остаток`**: `Остатки.КоличествоОстаток`
- `${Выборка.Поле}` — интерполяция представления объекта (ссылки, перечисления)
- `%{-Выборка.КоличествоОстаток}` — числовое значение; `-` инвертирует отрицательный остаток в положительную нехватку
- `\н` — перенос строки в строковом литерале XBSL
- `ИсключениеНедопустимоеСостояние(Описание = ...)` — именованный параметр

## Полный пример: Расход с контролем остатков

```xbsl
@Обработчик
метод ПослеЗаписи(До: Расход.Данные, ПараметрыЗаписи: Расход.ПараметрыЗаписи)
    знч НаборЗаписей = новый ТоварыНаСкладах.НаборЗаписей()
    НаборЗаписей.Фильтр.Установить(Регистратор = Ссылка)

    для Строка из Товары
        НаборЗаписей.ДобавитьЗапись(
            Период = Дата,
            ВидЗаписи = ВидЗаписиРегистраНакопления.Расход,
            Склад = Склад,
            Номенклатура = Строка.Номенклатура,
            Количество = Строка.Количество
        )
    ;

    НаборЗаписей.Записать()

    // Контроль: нет ли отрицательных остатков на этом складе?
    знч Запрос = Запрос{
        ВЫБРАТЬ
            Остатки.Номенклатура,
            Остатки.КоличествоОстаток
        ИЗ
            ТоварыНаСкладах.Остатки КАК Остатки
        ГДЕ
            Остатки.Склад == %Склад
            И Остатки.КоличествоОстаток < 0
    }

    исп Результат = Запрос.Выполнить()

    пер СтрокаСообщения = ""

    для Выборка из Результат
        СтрокаСообщения += "Не хватает товара ${Выборка.Номенклатура} в количестве %{-Выборка.КоличествоОстаток}\н"
    ;

    если не СтрокаСообщения.Пусто()
        выбросить новый ИсключениеНедопустимоеСостояние(Описание = СтрокаСообщения)
    ;
;
```

## Типичные ошибки (не делай так)

### ❌ ВидЗаписи у регистра Обороты

```xbsl
// НЕВЕРНО: ВидРегистра = Обороты, но указан ВидЗаписи
НаборЗаписей.ДобавитьЗапись(
    Период = Дата,
    ВидЗаписи = ВидЗаписиРегистраНакопления.Расход,  // ← ошибка компиляции
    Номенклатура = Строка.Номенклатура,
    Количество = Строка.Количество
)
```

```xbsl
// ВЕРНО: для Обороты — без ВидЗаписи
НаборЗаписей.ДобавитьЗапись(
    Период = Дата,
    Номенклатура = Строка.Номенклатура,
    Количество = Строка.Количество
)
```

---

### ❌ Фильтр после Записать()

```xbsl
// НЕВЕРНО: фильтр установлен после записи — старые движения не удалятся
знч НаборЗаписей = новый ТоварыНаСкладах.НаборЗаписей()

для Строка из Товары
    НаборЗаписей.ДобавитьЗапись(...)
;

НаборЗаписей.Записать()
НаборЗаписей.Фильтр.Установить(Регистратор = Ссылка)  // ← слишком поздно
```

```xbsl
// ВЕРНО: фильтр — сразу после создания набора
знч НаборЗаписей = новый ТоварыНаСкладах.НаборЗаписей()
НаборЗаписей.Фильтр.Установить(Регистратор = Ссылка)  // ← до всего

для Строка из Товары
    НаборЗаписей.ДобавитьЗапись(...)
;

НаборЗаписей.Записать()
```

---

### ❌ Суффикс Остаток в ДобавитьЗапись

```xbsl
// НЕВЕРНО: в ДобавитьЗапись суффикс не нужен
НаборЗаписей.ДобавитьЗапись(
    КоличествоОстаток = Строка.Количество  // ← ошибка: поля нет
)
```

```xbsl
// ВЕРНО: используем имя ресурса из YAML — без суффикса
НаборЗаписей.ДобавитьЗапись(
    Количество = Строка.Количество
)
```

---

### ❌ Контроль остатков ДО записи движений

```xbsl
// НЕВЕРНО: проверяем остатки до того, как записали расход —
// остатки ещё не обновились, проверка бессмысленна
знч Запрос = Запрос{ ВЫБРАТЬ ... ИЗ ТоварыНаСкладах.Остатки ... }
// ... проверка ...
НаборЗаписей.Записать()
```

```xbsl
// ВЕРНО: сначала записываем, потом проверяем
НаборЗаписей.Записать()

знч Запрос = Запрос{ ВЫБРАТЬ ... ИЗ ТоварыНаСкладах.Остатки ... }
// ... проверка и выброс исключения при необходимости
```
