# Паттерны: регистр сведений

Ключевые отличия от РН:
- Нет `ВидЗаписи` и нет `Регистратора`
- Фильтр — по **измерениям** (не по регистратору)
- `Записать(Замещать: Булево = Истина)` — управляет поведением при конфликте ключей
- `Период` в `ДобавитьЗапись` — **только** для периодических регистров (`Периодичность != Непериодический`)
- Виртуальная таблица для периодических — `.СрезПоследних()`

---

## Паттерн C1: Запись / обновление данных

Замещает существующие записи, попадающие в фильтр (поведение по умолчанию):

```xbsl
знч НаборЗаписей = новый <ИмяРегистра>.НаборЗаписей()
НаборЗаписей.Фильтр.Установить(<Измерение> = <Значение>)

НаборЗаписей.ДобавитьЗапись(
    <Измерение> = <Значение>,
    <Ресурс>    = <Значение>
)

НаборЗаписей.Записать()   // Замещать = Истина по умолчанию
```

Для **периодического** регистра — добавь `Период`:
```xbsl
НаборЗаписей.ДобавитьЗапись(
    Период      = <Дата>,
    <Измерение> = <Значение>,
    <Ресурс>    = <Значение>
)
```

## Паттерн C2: Добавить без замены существующих

`Замещать = Ложь` — новые записи добавляются, существующие по тому же ключу **не трогаются**:

```xbsl
знч НаборЗаписей = новый <ИмяРегистра>.НаборЗаписей()
НаборЗаписей.Фильтр.Установить(<Измерение> = <Значение>)

НаборЗаписей.ДобавитьЗапись(
    <Измерение> = <Значение>,
    <Ресурс>    = <Значение>
)

НаборЗаписей.Записать(Замещать = Ложь)
```

---

## Паттерн C3: Удалить записи по фильтру

Пустой набор с инициализированным фильтром удаляет все записи, попадающие в фильтр:

```xbsl
знч НаборЗаписей = новый <ИмяРегистра>.НаборЗаписей()
НаборЗаписей.Фильтр.Установить(<Измерение> = <Значение>)
НаборЗаписей.Записать()   // пустой набор = удаляет все по фильтру
```

## Паттерн C4: Срез последних (только периодический регистр)

Получить актуальное значение ресурса на текущий момент:

```xbsl
знч Запрос = Запрос{
    ВЫБРАТЬ ПЕРВЫЕ 1
        Таблица.<Ресурс> КАК Значение
    ИЗ
        <ИмяРегистра>.СрезПоследних() КАК Таблица
    ГДЕ
        Таблица.<Измерение> == %<Значение>
}
возврат Запрос.Выполнить().ПервыйИлиНеопределено()?.Значение
```

Можно передать дату среза параметром в `СрезПоследних()`:
```xbsl
<ИмяРегистра>.СрезПоследних(<Дата>)
```

## Типичные ошибки (не делай так)

### ❌ Забыть инициализировать фильтр

```xbsl
// НЕВЕРНО: фильтр не инициализирован — платформа выдаст ошибку
знч НаборЗаписей = новый ЦеныТоваров.НаборЗаписей()
НаборЗаписей.ДобавитьЗапись(Товар = ТоварСсылка, Цена = 100)
НаборЗаписей.Записать()
```

```xbsl
// ВЕРНО: Установить() без аргументов = пустой фильтр (все записи)
// Установить(Товар = ТоварСсылка) = фильтр по конкретному измерению
знч НаборЗаписей = новый ЦеныТоваров.НаборЗаписей()
НаборЗаписей.Фильтр.Установить(Товар = ТоварСсылка)
НаборЗаписей.ДобавитьЗапись(Товар = ТоварСсылка, Цена = 100)
НаборЗаписей.Записать()
```

---

### ❌ Указать Период для непериодического регистра

```xbsl
// НЕВЕРНО: Периодичность = Непериодический, но указан Период — ошибка компиляции
НаборЗаписей.ДобавитьЗапись(
    Период  = Дата.Сейчас(),   // ← нет такого поля у непериодического РС
    Товар   = ТоварСсылка,
    Цена    = 100
)
```
