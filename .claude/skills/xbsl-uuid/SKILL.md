# Генерация UUID v4

## Шаг 1: Сгенерируй UUID

```bash
python3 -c "import uuid; [print(uuid.uuid4()) for _ in range({N})]"
```

Где `{N}` — точное количество, переданное вызывающим скиллом.

## Если Bash недоступен

Генерируй UUID v4 самостоятельно: формат `xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx`, где `y` ∈ `{8, 9, a, b}`. Не используй счётчики, порядковые номера или повторяющиеся паттерны — каждый UUID должен быть уникальным.
